HandleSpaceRequired(IEnumerable<IViewElement>, OverlapType, Single)

Adds a tag to the currently active transaction indicating that there is not enough room to prevent overlap.

Declaration
public static float HandleSpaceRequired(IEnumerable<IViewElement> elements, OverlapType type, float parentSpaceRequired)
Parameters
Type Name Description
System.Collections.Generic.IEnumerable<IViewElement> elements

The elements for which overlap prevention is not possible. These elements should all belong to the same rooted parent, and should be under an active transaction.
OverlapType type

The type of overlap that is being prevented.
System.Single parentSpaceRequired

The additional height/width required to prevent overlap of the elements. This amount should be added to the height of the parent if we're preventing vertical overlap, or the width of the parent if we're preventing horizontal overlap.
Returns
Type Description
System.Single

The amount the height or width was adjusted to allow the elements to be placed without overlap.
