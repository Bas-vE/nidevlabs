EditorContexts

This collection tells Context Help which editors this content provider should appear in.

An empty collection is the same as specifying all editors.

The identifiers for the main 3 editor windows are: NationalInstruments.VI.Design.VIPanelControl NationalInstruments.VI.Design.VIDiagramControl NationalInstruments.Restricted.LabVIEW.VI.Designer.VIIconDocumentEditControl
