public AutoToolViewModel(IEnumerable<IDesignerToolViewModel> tools)
Parameters
Type Name Description
System.Collections.Generic.IEnumerable<IDesignerToolViewModel> tools

Set the sub tools that we manage. The order that they exist in the collection are the order they will be put in the visual tree. The standard tools must be added to the AutoTool in the following order for standard behavior:

  • NavigateTool
  • PlacementTool
  • WiringTool
  • TextTool
  • SelectionTool
  • ToolTipTool Deviation from this order may produce unpredictable results as the tools interact with each other. You are free to leave any of these tools out or add additional tools.
