Returns
Type Description
System.Threading.Tasks.Task<ILockedSourceFile>

A task representing the load of the referenced file. The result of this task is a disposable that will manager the lifetime of the referenced file. Disposing the returned ILockedSourceFile before the load completes will cancel the load operation and can prevent the source file reference from being added to the parent Envoy.
