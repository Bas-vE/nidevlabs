This method only merges HitLists from same domain. This method also take care of the duplicates that have the same offset.

Parameters

Remarks

It would throw SearchException if 1) the HitLists come from different categories, 2) The new HitList is ahead of the current one (HeadOffset is smaller). 3) the new HitList is null.
