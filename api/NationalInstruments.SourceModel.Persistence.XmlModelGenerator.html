Returns
Type Description
System.Collections.Generic.List<System.Object>

A list of cookie references (used for merge scripts)
Remarks

This method assumes the XmlWriter is positioned so that this method can call System.Xml.XmlWriter.WriteAttributeString(System.String,System.String) several times for each property on the object. We do this by calling NationalInstruments.DynamicProperties.DynamicPropertyExtensions.GetAllSetProperties(System.Object) on the object, looking at all properties whose symbols inherit from NationalInstruments.DynamicProperties.SerializablePropertySymbol, and invoking for all the ones that are persistent. If a property returns false for NationalInstruments.DynamicProperties.DynamicPropertyExtensions.IsPropertySet(System.Object,NationalInstruments.DynamicProperties.PropertySymbol), we do not write it out.
