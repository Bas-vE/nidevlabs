Interface IWiringBehavior

Clients should implement this interface to specify a custom type of wiring behavior for wires in their model; clients also can derive from one of the existing implementation to extend them if they mostly want the built-in behavior (e.g. ManhattanWiringBehavior, ArcWiringBehavior)
