Returns
Type Description

This method can give different answers when resizing from the top versus resizing from the bottom because chunks that have wired terminals should not be deleted. SetVerticalChunkCount(IVerticalGrowNode, Int32, GrowNodeResizeDirection)

Sets the VerticalChunkCount of a grow node.

Declaration
public static int SetVerticalChunkCount(this IVerticalGrowNode verticalGrowNode, int desiredChunkCount, GrowNodeResizeDirection direction)
Parameters
Type Name Description
IVerticalGrowNode verticalGrowNode

the vertical grow node on which to change the vertical chunk count
System.Int32 desiredChunkCount

The desired new vertical chunk count
GrowNodeResizeDirection direction

The resize direction, either Top or Bottom. Left and Right are not valid when changing the vertical chunk count.
Returns
Type Description
System.Int32

The actual new chunk count
Remarks

This method will add or remove chunks to the grow node in the specified direction. The desiredChunkCount is first coerced to a valid value, and then the chunks are added or deleted one at a time. Reasons for coercion include trying to grow too big or shrink too small, or trying to shrink past a chunk with wired terminals.
