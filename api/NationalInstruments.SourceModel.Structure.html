Syntax
public abstract class Structure : Node, IPropertyLocalStorage, IParsable, IQueryService, IViewElement, IWireShapingElement, IMergeSource, IWiringEnd, IViewElementParent, IMergeTarget
Remarks

A structure can have its own Terminals or may have Terminals associated with the Structure through BorderNodes. The Terminals associated with the Structure through BorderNodes allow connections between siblings of the Structure and Nodes within the Structure's Parent Diagram. A Structure is a namescope for Nodes. All BorderNodes within a Structure must have a unique name.
