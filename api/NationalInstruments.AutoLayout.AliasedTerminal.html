Class AliasedTerminal

This is an aliased terminal. Aliased terminal, as the name says is aliasing another terminal. The common usage of this is in Container node and Clusters. For example: The terminal on the border-node will be aliased as a terminal on the container node. This way algorithms can choose to see ContainerNode as any other TerminalNode
