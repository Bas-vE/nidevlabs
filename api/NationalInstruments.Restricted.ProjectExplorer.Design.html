ProjectTreeViewSelectionBehavior

Behavior class to bring a project itemViewModel into view, highlight the project itemViewModel, and synchronize selection. NOTE: Because of virtualization, project tree itemViewModel visuals are only created until they are visible in the tree view. NOTE: This helper class is used to bring the tree itemViewModel into view so its visual object is created. NOTE: Use case: NOTE: - Creating multiple new VIs in a row should always bring the VI project items into view NOTE: The new VI items are added to the bottom of tree. If we don't bring them into view, they will not be visible because the tree does not auto scroll.
