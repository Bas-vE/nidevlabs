SmartGuidesActiveProperty

Attached property meant to convey whether its object is actively showing the smart guides it allows. The intent of this property is to easily allow the toggling of smart guides drawing without having to muck with ALL of the various drawing options (i.e. if you want to turn off drawing, just change this to false rather than having to change ShowsLeftSnaplines AND ShowsRightSnaplines AND... etc.)
