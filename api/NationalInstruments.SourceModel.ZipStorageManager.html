Class ZipStorageManager

Storage manager which uses Zip technology to save everything in a project to a single zip file, typically with .lvprojectx extension Currently the only implementer is InMemoryZipStorageManagerBase which loads and manipulates the entire zip in memory, but additional derivations/strategies could be implemented as needed.
