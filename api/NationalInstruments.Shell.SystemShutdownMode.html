Enum SystemShutdownMode

The mode by which to exit the Windows session.

Namespace: NationalInstruments.Shell
Assembly: NationalInstruments.PlatformFramework.dll
Syntax
public enum SystemShutdownMode
Remarks

This exposes some but not all bits of the ExitWindowsEx EXW* flags.

Fields

Name Description
PowerOff

Shuts down the system and turns off the power. The system must support the power-off feature.
Restart

Shuts down the system and then restarts the system.
ShutDown

Shuts down the system to a point at which it is safe to turn off the power. All file buffers have been flushed to disk, and all running processes have stopped.
