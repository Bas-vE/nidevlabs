Class ConstConstraint

Represents a value that is guaranteed to be a constant at run-time. The actual value of the constant may or may not be known.

ValueIsKnown

Returns true if the constant value is known. It is sometimes possible to know that a value is constant but you may not be able to determine the actual value. So ValueIsKnown implies ValueIsConstant but not necessarily the other way around.
