ILocalizableContentService

This interface defines functionality useful for localization of Element and its NationalInstruments.DynamicProperties.PropertySymbols. Elements present localizable content via PropertySymbol values that have the NationalInstruments.DynamicProperties.PropertySymbolAttributes.Localizable attribute set. In many cases, this data is a simple string, but for PropertySymbols whose values are a more complex type, such as a PropertyExpression, an implementation of this interface may be used. This service also provides a way for Element to share its localizable property symbols and the values they store in localized string dictionaries.
