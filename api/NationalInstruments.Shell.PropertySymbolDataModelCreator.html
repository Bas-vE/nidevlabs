AttemptToCreate(IEnumerable<Element>, PropertySymbol)

This method checks the given list of models to ensure that every item on the list has the identified property and that the property is the same type on all the models. If this is true then the function returns a new PropertySymbolDataModel monitoring the given models for the identified property. If this is not true, the function returns null.
