Envoy associatedEnvoy

The Envoy that this service should be associated with. This method will be called once for each Envoy the service is associated with. If a new service is created for each Envoy, this method will only be called once, and this Envoy will match AssociatedEnvoy.

OnDetach(Envoy)

Called to detach this envoy service from its AssociatedEnvoy. This method can be called when the associated envoy is being removed from the project, or when the associated envoy changes in such a way that this service no longer binds to the associated envoy.

If the associated envoy is being removed from the project, it is possible that other envoy services bound to this associated envoy will have already been detached. Therefore, it is not safe to query for other services within this callback. Instead, this callback should only be used to clean up state that is local to and understood by this service.

Remarks

OnDetaching(Envoy)

Called before the service is detached from its AssociatedEnvoy. This method can be called when the associated envoy is being removed from the project, or when the associated envoy changes in such a way that this service no longer binds to the associated envoy.

If the AssociatedEnvoy is being removed from the project, or if the project is closing, all attached envoy services will receive this Detaching callback before any of them receive the Detach(Envoy) callback. This means is it safe to query for other services attached to AssociatedEnvoy from within this callback, but it is not safe within the Detach callback. This is a good point to perform any cleanup of this service.

OnModelClosed(Envoy)

Called when the model owned by this Envoy is closed. When this method is called, the model obtained through ReferenceElement is null.

Declaration
protected virtual void OnModelClosed(Envoy associatedEnvoy)
Parameters
Type Name Description
Envoy associatedEnvoy

The Envoy whose model has been closed. This is the same envoy that is passed to NationalInstruments.SourceModel.Envoys.EnvoyService.NotifyAttach(NationalInstruments.SourceModel.Envoys.Envoy) for envoy services which are instanced per envoy.
Remarks

This method is only called when the model referenced by the Envoy is closed while the envoy is still attached. This usually only happens when the last reference to the referenced model is released. When the last reference is released, the model is unloaded and can be loaded again later by calling LoadAsync(). If the envoy itself is removed from the project, this method will not be called.
