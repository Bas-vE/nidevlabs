Interface IModelGeneratorContext

The context for a should use for determining if a Connectable (Node or BorderNode) should generate wire connection information for its terminals.

Namespace: NationalInstruments.SourceModel
Assembly: NationalInstruments.PlatformFramework.dll
Syntax
public interface IModelGeneratorContext
Remarks

This interface is needed when generating fragments of a model that may be connected to wires that may or may not be generated along with the node.
