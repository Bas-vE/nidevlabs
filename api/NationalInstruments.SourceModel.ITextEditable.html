Interface ITextEditable

This interface is for model elements to implement such that UI adorners, which allow editing the text of our TextBlock's, can set the new value on the model.

Finish the current editing operation. This is not transacted if we never started a transaction (user never changed text). Checking result == cancelled can be used to know if you can do model changes here. You can also return False here to indicate that the final edit should be cancelled.
