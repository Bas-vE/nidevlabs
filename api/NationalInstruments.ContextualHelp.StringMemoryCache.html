Class StringMemoryCache

StringMemoryCache extends MemoryCache, and can be used when the values of a MemoryCache are strings. The class adds manual size estimation since the size estimate of MemoryCache is based on sized pointers which may be inaccurate and MemoryCache fails to trim cache when cache size is exceeded.
