public class DefaultSearchableService : ISearchable
Remarks

By default an element provides default information as above for search. You do not need to attach such service to an element if you do not want to customized the information. If you want to customize the information needed to search for an element, you should: Create a DefaultSearchableService instance; Change the property value you want to customize; Attache the service to the element.

Examples

Example elements attach SearchableService: BorderNode - Set Searchable to false. FrontPanel - ChildElements is FrontPanel.Components plus FrontPanelUnplacedItems.Components
