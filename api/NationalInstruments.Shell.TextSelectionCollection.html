Class TextSelectionCollection

public class TextSelectionCollection : IEnumerable<ITextOffsetRange>, IEnumerable
Remarks

When enumerating this collection, the active range will be returned first, if there is one. If there is no active range, the first item returned when enumerating will be one of the inactive ranges, if there are any.
