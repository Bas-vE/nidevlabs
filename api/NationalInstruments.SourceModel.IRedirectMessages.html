Interface IRedirectMessages

Use this interface to declare something that knows how to redirect a message from one source model element to another source model element.

Namespace: NationalInstruments.SourceModel
Assembly: NationalInstruments.PlatformFramework.dll
Syntax
public interface IRedirectMessages

Methods

RedirectMessage(MessageCreator, Element, String)

Source model elements call this function when they are told to report a message and they decide that the message would be better delivered to another source model element OTHER THAN a subpart of the source model element itself.

Declaration
void RedirectMessage(MessageCreator originalMessage, Element newRecipient, string newSubpartId)
Parameters
Type Name Description
MessageCreator originalMessage

The original message that was sent out. This will be used to populate the origination information about the first sender for the new message.
Element newRecipient

A source model element that will receive the message.
System.String newSubpartId

A string that is interpreted by the recipient as the subpart that the message is for (like a line number of a text code node)
