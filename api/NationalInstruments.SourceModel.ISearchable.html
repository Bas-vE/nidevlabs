SearchType

An identifier defines what we take as one kind of nodes in our search results. This is optional, if you do not want to customize classification you can ignore this, and the default value is ElementType. e.g. NumericControl and NumericIndicator have the same SearchType "Numeric", so they are taken as one kind of object when doing Find Object.
