AutomaticallyResolveUp

Returns true if the Linker should automatically resolve up through the scope and false otherwise.

Example 1: In the given hierarchy, if ChildScope automatically resolves up, ChildScope.Resolve("TargetEnvoy") will resolve to both :Project:ParentScope:ChildScope:TargetEnvoy and :Project:ParentScope:TargetEnvoy. But if ChildScope did not automatically resolve up, it would only resolve to :Project:ParentScope:ChildScope:TargetEnvoy.
