Class DataValueParserContext

Helper class to wrap the required information to parse custom data value. It is used with TryParseDataValue(DataValueParserContext, out Object) method. Contains owner Element, DataTypeReferenceTable, value NationalInstruments.DataTypes.NIType, and System.Xml.Linq.XElement.
