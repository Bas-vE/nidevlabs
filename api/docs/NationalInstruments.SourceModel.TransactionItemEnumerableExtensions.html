CollapseOwnerChanges<TComponent, TOwner>(IEnumerable<OwnerComponentChange<TComponent, TOwner>>)

Returns a collapsed enumeration of owner component changes. For example, an ownership chain of null -> Owner1, Owner1 -> Owner2 will get collapsed to null -> Owner2. In order to do this, the returned enumeration may contain new transaction items. The transaction items returned won't necessarily be the ones retained in the undo/redo stack of the transaction manager.
