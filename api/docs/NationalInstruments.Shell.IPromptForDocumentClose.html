Interface IPromptForDocumentClose

MEF Exportable interface for any operation/state that can prompt for cancel and/or cleanup on document close.

Namespace: NationalInstruments.Shell
Assembly: NationalInstruments.PlatformFramework.dll
Syntax
public interface IPromptForDocumentClose
Remarks

All IPromptRequests returned by this interface must include at least one IPromptResolutionOption: there must be some way for the user to close. Generally there should not be very many of these and/or it should be very rare to get multiple at once. There is not currently a guaranteed processing order, but we can consider that if more are added.
