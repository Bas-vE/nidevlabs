Enum ElementState

The state of a given Element.

Namespace: NationalInstruments.SourceModel
Assembly: NationalInstruments.PlatformFramework.dll
Syntax
public enum ElementState

Fields

Name Description
Attached

The state an Element enters when The Element derived object can walk up its Owner chain and eventually hit a DefinitionBoundary. The most common type of DefinitionBoundary is the SourceFileBase.

Content derived elements enter the Attached state during the call to Attach(IElementServices).
Detached

The state an Element enters when it is removed from the source model. The transaction for the removal has been committed, and the Element is still on the undo stack. This means that the Element can transition back to the Attached state by undoing the removal.

Content derived elements enter the Detached state during the call to OnDetached(IElementServices).
Detaching

The state an Element enters when it is removed from its Owner during a transaction, or when any Owner up the Elements Owner chain (within a DefinitionBoundary) is removed from its Owner. The Element stays in this state until the transaction in which the removal occurs is committed.

Content derived elements enter the Detaching state during the call to OnDetaching().
End

The state an Element enters when it has already been Detached, and it is now no longer present the undo stack. Elements in this state are not recoverable.
Start

The initial state of all Content derived Types. This is also the initial state of Element derived Types if the object is owned by some Content that is also in the Start state.
