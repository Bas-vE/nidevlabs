Class ReferencedFileDocumentType

The ReferencedFileDocumentType is a factory interface used to create documents and envoys for a file that exists on disk, but is not persisted as a SourceFileBase.

Envoy

The Envoy for which a document should be constructed. Normally, this is the same as the given Envoy. The return value of this method will be passed to the CreateDocument(Envoy) method on the IDocumentType corresponding to the returned Envoy.
Remarks

The given Envoy's ModelDefinitionType should match this document type's ModelDefinitionType. However, it is not required that the Envoy returned from this method also have that same model definition type.
