Class OwnerComponentCollection

An collection of Elements which automatically creates transactions and fires property change notifications; used as the data type of the Children property of Element and should be used by clients whenever they need a transacted collection of Elements in their model classes.

Move(Int32, Element)

Moves an item to a new index in the collection. This is effectively the same as calling Remove(item) followed by Insert(index, item). However, internally, the move is optimized in that it won't trigger certain callbacks due to items being removed and added and generally will result in fewer transacted changes.
