Interface IDocumentEditControlInfo

Provides information and acts as a factory for DocumentEditControls When a document is created the list of edit control is needed to provide user interface for the edit controls. This interface provides those interface pieces. When the user switches to a particular editor for a document then the DocumentEditControl is created through this interface. This allows on demand creation of the document editors (DocumentEditControl).
