Struct EnvoyAndLoadResult

A custom result returned from APIs that synchronously add an Envoy to a project, and then immediately begin the load of that Envoy. This result contains the Envoy that was added synchronously, and a Task representing the load of that Envoy. This enables callers of APIs that do this type of work to transact around the add of the Envoy, but not around the load of the Envoy.
