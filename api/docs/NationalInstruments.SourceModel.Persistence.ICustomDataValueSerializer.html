Interface ICustomDataValueSerializer

ICustomDataValueSerializer should implemented for for all custom data value serializers. Should provide query support using ITypeServiceProvider and keywords so that CustomDataValueSerializationManager can import ITypeServiceProviders and get custom data value serializer for the given type.

ParsableNamespaceVersionCompatibility

Returns the namespace version compatibility requirements introduced by this ICustomDataValueSerializer. The NationalInstruments.DynamicProperties.ParsableNamespaceVersionCompatibility.OldestCompatibleVersion returned will be merged with the NationalInstruments.DynamicProperties.INamespaceSchema.OldestCompatibleVersion, and the greater of the two versions will be used for compatibility purposes. The resulting oldest compatible version should guarantee that no mutations can modify the content persisted with this serializer if it was persisted between the resulting oldest compatible version and the NationalInstruments.DynamicProperties.INamespaceSchema.Version.
