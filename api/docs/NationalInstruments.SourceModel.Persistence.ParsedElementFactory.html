Class ParsedElementFactory

This class helps during parsing of dynamic elements by finding the proper factory method and calling it, as well as handling caching of the factory methods for best performance.

Constructors

ParsedElementFactory(ICompositionHost)

This version of the constructor calls into System.ComponentModel.Composition.Hosting.CompositionContainer to obtain the list of factory method exports using ParserContract as the contract identifier.

Declaration
[ImportingConstructor]
public ParsedElementFactory(ICompositionHost host)
Parameters
Type Name Description
NationalInstruments.Composition.ICompositionHost host

Host used to obtain parser factory method exports and to use to create an IElementCreateInfo to pass as an argument to all factory methods.
Remarks

This constructor is expensive (could take up to a couple seconds to run in a large system with 1000s of factory methods) so should be avoided during startup. The expensive part is the call to GetExports() on System.ComponentModel.Composition.Hosting.CompositionContainer. This call is not involved in the other form of the constructor, so that form us much less expensive.
