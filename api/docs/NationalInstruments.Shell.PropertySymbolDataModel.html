Class PropertySymbolDataModel

PropertySymbolDataModel is used to track a single property on a set of Element objects, to notify callers (such as the ribbon bars) when those properties change and to provide a single point of binding for writing back to those objects.

UpdatePropertyValueIfNeededCore()

This function is only invoked when the property values definitely need to be recalculated. Derived classes may override this to update the values of any properties they define which change when the values of the selected elements change. Anyone overriding this should likely be overriding RaisePropertyChangedCore as well to raise property changed events for the child's properties.
