Remarks

Specify a namespace if you want or need the ability to have mutations run on the envoy that manages your definition. Specifying a namespace will also help ensure your definition won't collide with other definition in the system.

Regarding images in this attribute: It is assumed at this time that all image resources are in the same assembly (i.e. typeForImageResources).
