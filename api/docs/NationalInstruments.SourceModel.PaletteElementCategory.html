Remarks

For example, modes commands API calls CalculateDisplayElements(false) to disable collapsing behavior, because otherwise 'ModeLeaf3' in below hierarchy Root |- Function A (Mode Category) |- ModeLeaf1 |- ModeLeaf2 |- Function B (Mode Category) |- ModeLeaf3 will be promoted in CalculateDisplayElements() as follow: Root |- Function A (Mode Category) |- ModeLeaf1 |- ModeLeaf2 |- ModeLeaf3 and generates wrong modes commands hierarchy for mode selector.
