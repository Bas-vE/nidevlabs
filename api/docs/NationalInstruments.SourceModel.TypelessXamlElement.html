Remarks

NOTE: the namespaces represented by WpfNamespace and XamlNamespace will automatically be defined with prefixes "wpf" and "x"; there is no need to suggest them.

This method only guarantees that the namespace specified will be defined with some prefix, not necessarily the one passed in. Do not assume that you will be able to use the specified prefix to refer to the namespace.
