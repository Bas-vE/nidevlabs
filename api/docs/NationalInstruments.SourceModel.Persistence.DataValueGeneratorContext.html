Class DataValueGeneratorContext

Helper class to wrap the required information to serialize custom data value. It is used with TryGenerateDataValue(DataValueGeneratorContext) method. Contains IElementPropertyGenerator, NationalInstruments.DataTypes.NIType of data value, and data value.
