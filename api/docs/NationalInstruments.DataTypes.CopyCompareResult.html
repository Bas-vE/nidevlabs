Enum CopyCompareResult

Possible results of the CompareTypesForCopy(). If you are changing this enum please make sure that you update the two helper static methods. Callers assume that this enum is in severity order. That means Identical must be first, then any coercions/conversions, then any conflicts. If you add to the enum, you must insert new items in the correct location.
