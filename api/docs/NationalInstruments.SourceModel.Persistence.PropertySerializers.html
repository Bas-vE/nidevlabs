NullDifferentiatingStringSerializer

A string serializer suitable for XML. This serializer will escape white space characters as appropriate so white space is guarnateed to be preserved across various XML parsers. If preservation of white space isn't required for the string, the serializer acts as a simple "pass through" serializer.

StringSerializer

A string serializer suitable for XML. This serializer will escape white space characters as appropriate so white space is guarnateed to be preserved across various XML parsers. If preservation of white space isn't required for the string, the serializer acts as a simple "pass through" serializer.

Treats values of System.String.Empty and null as equivalent. If the NationalInstruments.DynamicProperties.SerializablePropertySymbol's default value is System.String.Empty or null, the symbol will not be considered set if the value is either System.String.Empty or null. Empty/null strings will always be parsed as System.String.Empty, and will never result in null.
