AddToProject(Boolean)

Adds this itemViewModel to the project. This function return a folder project itemViewModel if itemViewModel is a folder. This function returns an incomplete project itemViewModel (has no unique id) if it is a file because adding a file to project is async. The Id is assigned asynchronously when the file envoy is added to the project.
