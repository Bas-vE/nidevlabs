Class NotifyPropertyChangedTransactionItem

Transaction item that is used to notify of a change to a property and is optimized for storage. This transaction item doesn't store new/old values and doesn't do anything on Rollback/Recommit. It exists purely for triggering notifications through the ModelNotificationManager.
