FindPreferredEnvoys(Envoy)

Find the preferred envoy typically for editing purposes. This will look at the provided envoy and its links when determining which envoys should be considered preferable. This will divide the envoys into tiers of preference and will return the most preferred tier.

Envoys in a tier are also ordered. In a given tier, the original envoy is first (assuming it belongs in the tier), followed by envoys referencing loaded files, followed finally by the rest of the envoys in the tier.

Tiers are created giving preference to:

  • Envoys with already open documents
  • Envoys under supported targets (IsSupportedUnderScope(Envoy))
  • Envoys in the last active target

The actual preference algorithm can be seen in NationalInstruments.Shell.DocumentExtensions.PreferredEnvoysTier.CalculateRank(NationalInstruments.Shell.IDocumentManager,System.Collections.Generic.HashSet{NationalInstruments.SourceModel.Envoys.EnvoyManager},System.Collections.Generic.HashSet{NationalInstruments.SourceModel.ITargetScopeService},NationalInstruments.SourceModel.Envoys.Envoy).
