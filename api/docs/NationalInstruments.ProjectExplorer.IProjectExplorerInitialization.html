HiddenFilesAndFolders

Gets a list of file extensions, relative folder paths, and folder names that should not be visible in the project explorer. For example, ".lvcodedb", ".lvprojectcache", LongPath.Combine(Folder, SubFolder). To blacklist a folder name, prepend the name with '', i.e., "CNode" hides all folders named "CNode".
