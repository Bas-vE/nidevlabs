Class DesignerEditControl

The DesignerEditControl is the root visual editor object used to edit a PlatformFramework based editing surface. It supposed both Diagram based editing as well as XamlElement based editing. The DesignerEditControl owns the visuals for the editor surface as well as the tooling used to manipulate the objects on the design surface.
