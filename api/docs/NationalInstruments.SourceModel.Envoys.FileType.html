Class FileType

The base class for most implementations of IFileType. Clients are not intended to inherit from this class directly. Instead, there are multiple derived classes such as SourceFileType, and ReferencedFileType that should be derived instead.

public EnvoyAndLoadResult AddEnvoyForStream(Element parent, EnvoyCreateInfo createInfo)
Parameters
Type Name Description
Element parent

Indicates the suggested Element under which the new envoy should be added.
EnvoyCreateInfo createInfo

The EnvoyCreateInfo describing how the envoy should be created.
Returns
Type Description
EnvoyAndLoadResult

An EnvoyAndLoadResult containing the Envoy that was newly added, and the System.Threading.Tasks.Task<TResult> representing the load of the newly created envoy from the given stream. The ILockedSourceFile returned from the Task is for the newly created and added envoy. This object controls the lifetime of the in memory representation of the referenced file in memory. When the last outstanding instance of this object is disposed, the referenced file in memory will be released and eligible for garbage collection, and the envoy will need to be re-loaded in order to access the contents of the file.
Remarks

When this method is called, there is not necessarily a file on disk that is associated with the Envoy. Thus, implementers of this method must immediately call BeginLoadFromStream(Stream, EnvoyLoadOptions, PlatformDispatcher, Action<ILockedSourceFile, Exception>) to load the referenced file from the stream. The file may be given a storage path and saved at a later time.
