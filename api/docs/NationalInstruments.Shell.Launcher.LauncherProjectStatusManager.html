Class LauncherProjectStatusManager

This class manages the status of Launcher projects. New projects, those that have been installed in the last NationalInstruments.Shell.Launcher.LauncherProjectStatusManager.ExpiredTime, are marked as "New" in the Launcher. This class is singleton.
