Struct DateTimeSignature

Class that wraps a System.DateTime. The purpose of this class is to centralize the comparison of timestamps. Because there is no exposed method to access the System.DateTime.Ticks of the wrapped System.DateTime, this prevents clients from incorrectly comparing timestamps with exact precision. The accuracy of timestamps varies in multiple ways. E.g. when a file is created on a FAT file system, the timestamp is rounded up to the nearest even second. When a file is packaged and installed by nipm, the timestamp is cut off at seconds. Because of this, it is not safe / correct to compare them exactly. The Equals method of this class considers timestamps that are within 2 seconds of each other to be equal. This has been determined to be correct for all scenarios we are aware of.
