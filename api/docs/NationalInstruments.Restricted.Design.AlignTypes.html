Enum AlignTypes

An enum used in the manner of a bit array. The AlignTypes is a blueprint meant to contain information about every way in which two or more elements are aligned. It's used as a bit array because it is possible to be aligned in multiple ways (a very common example of this would be two identically sized elements lining up vertically, which would make them left aligned AND right aligned AND horizontal center aligned.
