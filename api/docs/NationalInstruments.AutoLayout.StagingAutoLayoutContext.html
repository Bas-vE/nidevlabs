Class StagingAutoLayoutContext

This is the context class used by the algorithms. This is the critical piece for staging multiple algorithms. The original context classes are created at sourcemodel levels, and this LIR context can be created only by having an original context.

SaveAdditionalNodeInformation(TerminalNode)

Save any additional node information required when we map to-and-from nodes (ex: SyncNodeDataFromOriginal, SyncToOriginalData) These are the information which can not be used in isolation (of one node) and we cannot map right now because parts of the information may not yet be ready. For example we save the node pairing information here. The reason is, pairing information involves two nodes. It may be the case that the other node is not yet mapped.
