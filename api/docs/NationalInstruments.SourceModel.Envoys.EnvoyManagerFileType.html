Class EnvoyManagerFileType

An implementation of SourceFileType specifically for file types than contain an EnvoyManagerFile. This will create a model on the lines of: SourceFileReference - Name="Foo.xyz", BindingKeywords="..., EnvoyManager, ..." SourceFile XyzDefinition EnvoyManagerFile [Optional: if CreateEmbeddedNameTrackingScope] NameScopingEnvoy - Name="Foo.xyz", NameTracksFileName=True, AutomaticallyResolveUp=True
