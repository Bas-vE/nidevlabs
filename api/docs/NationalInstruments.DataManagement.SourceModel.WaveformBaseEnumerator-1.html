Class WaveformBaseEnumerator<T>

Remarks

The validity check of the input may be a spot check, meaning the function may return a non-null value, yet the input is not correct. This was done in order to make import more efficient, doing as little reading in as possible. Check the output of the deserializer for null when using it, and if it returns null, you should continue to a new factory.
