Syntax
public abstract class Diagram : Wireable, IPropertyLocalStorage, IParsable, IQueryService, IViewElement, IWiringEnd, IMergeTarget, IViewElementParent
Remarks

The Terminals on a Diagram come from the Terminals on BorderNodes in the Structure that is the parent of the Diagram. The Diagram is a namescope for Nodes. All Nodes in a Diagram must have a unique name.
