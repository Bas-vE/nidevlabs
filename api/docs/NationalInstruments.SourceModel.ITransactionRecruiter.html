Interface ITransactionRecruiter

Interface that is used to record transacted changes in the source model. Source model elements should interact directly with this interface and the extension methods defined in this file to record changes being made to them such that undo/redo work properly. Entities outside of the source model should use the ITransactionManager interface to begin and end transactions that will change the state of the source model.

DisableTransactionRecording()

Temporarily disables recording of transactions until the returned IDisposable is disposed. If called multiple times, all IDisposables must be disposed before transaction recording will resume. This function isn't thread safe. If callers dispose from a different thread than the IDisposable was created from, it's up to the caller to ensure proper usage and synchronization.
