GuessProjectExplorerViewModel(Project)

NOTE: You should really never be calling this function. We can't guarantee correct behavior from this function. If you have an edit site, please call GetProjectExplorerViewModelFromEditSite. Attempt to find the project explorer view model from the project. Normally you should get this from the visual side by getting the edit site, toolwindow and pulling it from the DataContext. However, if you don't have access, call this and it will attempt to give it to you. It might not be the right one however. There may be folder synchronization issues, but it should work. If it can't find one, it will spin up a new view model as long as it can access the model.
