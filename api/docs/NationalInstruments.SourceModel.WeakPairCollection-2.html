Class WeakPairCollection<TKey, TValue>

A container for mappings between a key and a value, where the key is allowed a single value. A weak reference is maintained to the key, and the lifetime of the value is tied to that of the key. A strong reference to the value is only held while there is an external strong reference to the key. If the key is garbage collected, the value is eligible for garbage collection as well - as long as there are no external strong references.

This class is very close in spirit to the System.Runtime.CompilerServices.ConditionalWeakTable`2, but allows for enumerating over the pairs that have been added to the collection. If you do not need this enumeration capability, you should consider just using System.Runtime.CompilerServices.ConditionalWeakTable`2 instead.
