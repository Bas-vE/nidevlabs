NeedsTypeConversion(NIType)

Gets whether original type needs to be converted to a value other than DefaultType if it is not compatible with this type. For example, when configuring a fixed point type and a complex fixed point type is passed in, we want to do a conversion to maintain parts of the original type (like word length, integer bits, and fractional bits). If this method returns true, ConvertType will be called.
