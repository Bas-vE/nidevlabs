Class HighTowerOptions

Class which stores the options for the algorithm. These options also act as heuristics to tweak the algorithm. Default values: BlockHorizontalCushion = 3; BlockVerticalCushion = 3; WireHorizontalCushion = 2; WireVerticalCushion = 2; MaxTimeToRun = 200; MaxExtrapolationCount = 4; SnapToGridSize = new SMSize(7, 7); AlternateSeedPointPenalty = 0; BendCost = 90; CrossingCost = 3; TranslucentObjectCrossingCost = 15; HorizontalBackEdgePenalty = 10; VerticalBackEdgePenalty = 2; MaxNumberOfBendsDuringMerge = 3; MaxNumberOfBendsDuringBranching = 3; LastBendAffinityCushion = Math.Min(SnapToGridSize.Width * 4, SnapToGridSize.Height * 4); MaxNumberOfRoutesToBeFound = 50; MaxTimeDuringCleanUp = 200; MaxTimeDuringLive = 30;
