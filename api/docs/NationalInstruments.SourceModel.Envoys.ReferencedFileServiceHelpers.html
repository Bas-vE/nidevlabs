MakeProjectRelativeUri(IReferencedFileService)

Make a URI based storage path that the storage manager can use to resolve to a file can be resolved to a file using a project. This will generate a string URI path on the lines of the following. In this example, the [] groupings and non-escaped strings are here in order to make the strings more human readable. niAggregate://Path / [niPackage://NI/Analysis/Basic/1.0.0.0] / [niFile://Filtering] / [niFile://ButterworthFilter] / [niFile://Bandpass Filter (DBL).gvi] The actual fully escaped URI path returned would look more like the following. niAggregate://Path/niPackage%3A%2F%2FNI%2FAnalysis%2FBasic%2F1.0.0.0/niFile%3A%2F%2FFiltering/niFile%3A%2F%2FButterworthFilter/niFile%3A%2F%2FBandpass%20Filter%20%28DBL%29.gvi
