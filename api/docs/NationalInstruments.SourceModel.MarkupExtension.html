Remarks

Rules:

  1. A markup extension is of the form { identifier attribute* }.
  2. An attribute is of the form identifier | identifier = value, where value is an identifier or a markup extension.
  3. Identifiers contain letters and periods and may be qualified by a namespace prefix (separated by a colon).
