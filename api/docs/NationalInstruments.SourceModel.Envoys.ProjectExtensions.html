CanMoveProjectToDestination(Project, String, out ICollection<String>)

Determines whether or not a project and its associated files can successfully move from its present location to some destination. This will verify that all envoys in the project with file services will be able to successfully be opened after the move. If the main project file cannot be moved, the return result will indicate why and the list of invalid paths will be empty. If invalid paths contains entries, then the project can be moved, but the files associated with the project will cause problems.
