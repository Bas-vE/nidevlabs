ReplacementsPaletteKeyword

Replace palette keyword. This is generally used just like modes palette, but not specifying a mode, instead the parent element of a leaf should have an Element id that if you strip .Modes off the end will lookup the corresponding category in real palette for replacement. For example: Cluster terminal does not exist in regular palette Cluster terminal is in Replacements palette with parent NI.LabVIEW.VI:PaletteCategory.Cluster.Modes
