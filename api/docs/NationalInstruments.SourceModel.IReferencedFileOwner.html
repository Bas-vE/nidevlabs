Interface IReferencedFileOwner

Object that owns the contents of a referenced file. It is able to receive an object representing the in memory representation of their referenced file, and attach it to whatever their model is. The type of the object received is determined by the owner's implementation of ReferencedFileStorageHelper<T>, where T is the type of the object.
