GetTerminalIdentifierFixupsForMerge()

Returns a list of fixups that should be applied to wiring operations during a merge operation. In this case, any wire in the merge operation that is connected to the wireable with the given ContentId will have its terminal identifier updated with the value supplied as the string in the dictionary. This can be useful when creating a new structure diagram from an existing structure diagram if the border nodes are not part of the copy selection.
