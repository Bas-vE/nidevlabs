OldName

Gets the name of the Envoy this dependency was last resolved to. For a dependency that has never resolved, this will match the TargetName property of the dependency. If TargetName is an absolute name, this property will return the absolute qualified name the dependency was last resolved to. Otherwise, the property will return the name returned by calling MakeRelativeDependencyName(Envoy) on the envoy the dependency was last resolved to.
