ProhibitCookies

Indicates whether or not the provided merge script should be capable of being used by other processes or not. Set to True if you need to inflate the merge script in another process or persist the merge script for later use.

While a merge script that is compatible across processes is more portable, it is also more expensive to generate and inflate. Set to False when you don't care about portability and want the fastest possible performance.
