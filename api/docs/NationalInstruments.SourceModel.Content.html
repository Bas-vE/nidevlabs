ReportMessage(MessageCreator, String)

Creates a new message and adds it to the element GetMessages() collection. It translates "this" into the appropriate sub element based on the value of subpartId. Often this is used to report an error against a specific terminal of a node (terminals do not have their own unique identifier). Some elements may use the subpartId to change the error message to report parts of themselves that are not actually elements, such as the row and column number of a block of text in a text node. The interpretation of subpartId is entirely in the hands of the element, which is why this function is virtual.
