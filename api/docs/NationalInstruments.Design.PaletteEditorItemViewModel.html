AddCustomConfigurationHeaderContent(ICommandPresentationContext)

The .gpal document customizes its configuration pane with this function, because top-level category's right-rail header needs to be treated differently:

  1. Use non-editable static text for top-level category's Display(topCommand), use text box for other categories' Display command(edits allowed)
  2. Do not show 'Edit Icon' button for the top-level category.
