Syntax
public abstract class SimpleTunnel : Tunnel, IPropertyLocalStorage, IParsable, IQueryService, IViewElement, IWireShapingElement, ICompositeWireJoint, IEnsureOwnView
Remarks

A SimpleTunnel can be used with a SimpleStructure to connect diagrams related to the Structure. If you just need this class with no additional functionality, create a child class. This class is not used as a concrete class itself in order to leave space for future functionality adjustments.
