Interface IObserveContent

Interface for anyone that wants to observe when an Content leaves the model. Usually this is used by objects that hold on to references to an Content to know when that reference becomes invalid. Registration is done via RegisterObserver(Content, IObserveContent). The IdentifierDictionary can be found in IdentifierDictionary.

ObserverOwner

Owner of this observer. If the owner leaves the model, the observer is automatically unregistered in a transactional way. You may return null in which case no automatic unregistration is done.

Called when the content leaves the model. This is guaranteed to be called at a time when it is safe to make transacted changes. You should avoid making non-transacted changes from this call since it is not run on undo/redo.
