System.Collections.Generic.IList<NationalInstruments.Core.BindingKeyword> addKeywords

This list contains any keywords that other IInitializeEnvoy services have indicated should be added. Any new keywords added by this service should be added to the end of this list.
System.Collections.Generic.IList<NationalInstruments.Core.BindingKeyword> removeKeywords

This list contains any keywords that other IInitializeEnvoy services have indicated should be removed. Any keywords removed by this service should be added to the end of this list.
System.Boolean newDefinition

True if this envoy is being initialized with a new definition and False otherwise (e.g. definition is being loaded from disk).
Returns

The model definition type that is overridden by this envoy or BindingKeyword.Empty if this envoy doesn't override the model definition type.
Remarks

If the same keyword shows up in both the addKeywords and removeKeywords list, the keyword won't be applied to the envoy.
