Interface IUserPreferencesPage

Interface for exporting a User Preferences page. Implement this interface and export it to add a page to the User Preferences Dialog. See the documentation for IUserPreferencesPageInfo for an explanation of the required metadata.

Each user preferences page is displayed separately in the user preferences dialog. It is the responsibility of the page to save user preferences for itself
