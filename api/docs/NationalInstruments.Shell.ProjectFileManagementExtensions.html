AddExternalFileAsCopy(Project, IQualifiedScopeService, FileStreamInfo, Action<ILockedSourceFile, Exception>)

Called to create an external file as a stream (then copied into the project's storage upon save). The actual external file definition will be stored under the NullTarget if one exists, otherwise it will be added to targetToAddUnder. If there is a NullTarget, then we will clone an EnvoyReference under targetToAddUnder and return that.
