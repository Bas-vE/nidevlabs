Interface IVerticalGrowNode

Implement this interface to make a node that can grow by vertical chunks (a chunk is a group of terminals). The VerticalChunkCount can be set directly through the interface, or, if the VerticalGrowNodeBoundsRule is registered as a ModelBatchRule, it will be set when the node height is changed

Remarks

This property must enlist a property item when this value changes. The implementing class must provide both the Get and Set methods, but the preferred method for setting this property is by calling the SetVerticalChunkCount(IVerticalGrowNode, Int32, GrowNodeResizeDirection) extension method, as this provides additional checks that ensure it is okay to add or remove the desired chunks, and it adds and removes the chunk terminals when the chunk count changes.
