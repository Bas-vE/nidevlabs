GetContext(IViewModel)

Gets the context for the given view model. Does not modify the SelectionContext. This is useful for getting content for sub selections (like the properties of a graph's axis). This method must be used with caution, as the commands returned by it might be relying on the actual selection (by getting it from the host). For example, in the Execute for one of the commands we might be assuming that the viewModel passed in here is part of the selection. In general, if you're using this method you should make sure the commands returned by this won't make any assumptions.
