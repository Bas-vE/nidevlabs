Enum SearchOptions

Options available for search

Namespace: NationalInstruments.SourceModel
Assembly: NationalInstruments.PlatformFramework.dll
Syntax
[Flags]
public enum SearchOptions

Fields

Name Description
Default

Default Search options Ignore case and partial word match and skip VI.Lib (i.e. source files that are automatically added to the project) and load source files if necessary
IncludeVILib

Search include VI.Lib and source files that are automatically included within the project If necessary, they will be loaded to search for dependencies TODO : TBD ??? Include external libraries or Mocs that are implicitly/automatically added?
MatchCase

Ignore the case while searching dependencies based on name Not applicable when searching the dependencies that are resolved to given Envoy

MatchQualifiedName

Query matches target QualifiedName, ignoring name's AutomaticallyResolveUp property. It will ignore MatchCase, MatchWholeWord, RegularExpression.
MatchWholeWord

Entire word must match while searching the dependencies based on name Not applicable when searching the dependencies that are resolved to given Envoy

OnlyInMemorySourceFiles

Limit the search to only those source files that are currently loaded in memory At present it is mainly used for testing and performance benchmarks but it can be exposed to end user if found to be useful
RegularExpression

Given string is a regular expression When this option is selected, MatchCase and MatchWholeWord must not be selected Not applicable when searching the dependencies that are resolved to given Envoy TODO : To be implemented. At present it will throw NotImplemented exception
