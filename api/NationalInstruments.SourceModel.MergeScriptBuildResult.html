Class MergeScriptBuildResult

The result of building a merge script If the merge script was InMemoryOnly then the Script contains the elements and ParseTables contains the referenced tables (not serialized) If the merge script was FullText then the Script contains both the elements and the serialized tables and ParseTables will be null
