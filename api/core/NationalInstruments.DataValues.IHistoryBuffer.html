Interface IHistoryBuffer

Base interface to a circular buffer of data. It allows access to the most recent data and a total count to determine how much data has come before it.

Verifier

A verifier that can be used by clients to tell if previous data value are unchanged. When code inspects a buffer it can record the verifier. The next time it gets a buffer, if the verifier is unchanged then it is safe to assume all data up to the previous total count is unchanged. If the verifier is different, then the entire buffer should be treated as new.
