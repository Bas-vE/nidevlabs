Remarks

Goal is to emulate the contract of .NET 4.62 TaskAwaiter.GetResult, as implemented via ThrowForNonSuccess. We can't implement that directly since don't have direct access to internal Task APIs (e.g. IsRunToCompleteion or GetExceptionDispatchInfos), so detect equivalent states here and actually call GetResult in a general catch, then capture that into this.TaskExceptionInfo. That means that our awaiter's GetResult will throw during continuation. Reference implementation: https://referencesource.microsoft.com/#mscorlib/system/runtime/compilerservices/TaskAwaiter.cs,ca9850c71672bd54
