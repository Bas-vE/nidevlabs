Number is an arbitrary precision library capable of implementing arithmetic operations exactly on base 2 numbers, up to a maximum precision bound. With the exception of NaN, Number significands are assumed to be LSB-standardized, with no trailing zeroes. NaN payloads mirror that of the corresponding IEEE-754 format and may contain trailing zeroes.
