Class NIMessageBoxWindow

General purpose NI message box utility. Most of the optional (or less frequently used) properties can be set using either the verbose version of the constructor or public property setter methods.

The typical usage of this class is to:

  1. Create an instance of this class
  2. Optionally set properties
  3. Show the message
  4. Optionally get the user selection (Result)

SetDoNotShowKey(Type, String)

Sets a qualifier and key for saving the "Don't show this message again" selection as a user preference. Setting the key implies that the "Don't show..." check box will be displayed (if it has not been previously checked). If the check box was checked when this message (as identified by the qualifier and key) was previouslky displayed, the ShowModal() method will immediately return with the Result property set to the previous selection rather than actually displaying the message dialog to get a Result.

Remarks

The default mode of operation of the dialog is to favor fitting the message on as few lines of text as possible, putting an upper bound on the text area's width as specified by MessageTextMaxWidth. Use this method to override this behavior and keep the dialog a fixed width.
