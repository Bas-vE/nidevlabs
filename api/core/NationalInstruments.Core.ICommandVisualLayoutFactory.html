Interface ICommandVisualLayoutFactory

This interface allows visuals to be reused and organized in the UI. A visual layout can be left in the visual tree when not needed by calling Detach on its visual factory to disconnect it from the command that is going out of scope. When the visual layout is needed again, just call Attach with the command that is coming into scope.
