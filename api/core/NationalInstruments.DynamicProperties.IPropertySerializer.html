ParsableNamespaceVersionCompatibility

Returns the namespace version compatibility requirements introduced by this serializer. The OldestCompatibleVersion returned will be merged with the OldestCompatibleVersion, and the greater of the two versions will be used for compatibility purposes.
The resulting oldest compatible version should guarantee that no mutations can modify the content persisted with this serializer if it was persisted between the resulting oldest compatible version and the Version.
