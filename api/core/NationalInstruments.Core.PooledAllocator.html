CreateMultithreadedPool<T>(Int32, Func<T>, Action<T>)

Create a multi-threaded pool. It is legal to use this pool from any number of threads, and controls allocations and deallocations with a mutex.

Int32 initialSize

Initial size of the pool (this many objects are allocated up front). If zero is supplied, no objects are allocated up front, but they will be allocated on demand in chunks increasing in powers of 2.
