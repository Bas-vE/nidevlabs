ParseDynamicPropertiesForElement(IParsable)

Parses properties for the specified Element, but doesn't recurse and parse any child components that have been persised in the stream. If persisted child elements are present in the stream, the parser will skip over them without processing them and leave the stream in the same state as if ParseDynamicElement was called.
