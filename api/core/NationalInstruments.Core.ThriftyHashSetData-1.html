Struct ThriftyHashSetData<TValue>

A HashSet like collection which is optimized for storage of a few number of elements. When the HashSet contains fewer than 7 elements it will used a optimized storage. When there are greater then 7 elements a standard HashSet is used. When there are 0 or 1 elements there is not allocation.

NOTE: A ThriftyHashSet is a !DANGER! mutable struct !DANGER!. You cannot pass it around, it must remain a member of another class which provides methods and properties to access the hash set.
