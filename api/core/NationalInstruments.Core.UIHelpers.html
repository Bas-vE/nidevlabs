Class UIHelpers

This class is home for static methods that might be useful in more than one place, that relate to using the UI side of the SourceModel, and that don't have a natural home in another class. The purpose of this class is to encourage factoring out of duplicated code. There is also a similar ModelHelpers class.

TryGetPosition(MouseEventArgs, UIElement, out Point)

Try to get the mouse position in relativeTo's coordinate system. This was created because our mouse event routing on Silverlight (in DesigntimeCanvas) seems to occasionally throw exceptions when we try to get mouse position on something that is no longer rooted. We haven't figured out a fast way to tell if an item is rooted and not send events to such objects (just checking visual tree vastly slowed things down).
