Type Description

Remarks

This function starts with the given culture and executes the provided getResourceContainer delegate so it can determine if the desired data can be located. The specific knowledge of what kind of resource container is entirely encapsulated in the provided delgate. If getResourceContainer returns null, the parent culture of the culture passed to the delegate will be sent. This continues until the invariant culture is reached. As soon as CultureInfo.InvariantCulture is reached, or a non-null value is returned by getResourceContainer, the search ends.

FindNeutralCulture(CultureInfo)

Finds the neutral culture for the given CultureInfo.

Declaration
public static CultureInfo FindNeutralCulture(this CultureInfo culture)
Parameters
Type Name Description
CultureInfo culture

The culture to check. If null, then CurrentUICulture will be used.
Returns
Type Description
CultureInfo

culture, if it is a neutral language already. Else, the closest neutral language parent of culture. Null if there is no neutral language parent.
Remarks

Certain languages like Chinese have multiple layers of neutral languages. For example, Chinese is the neutral language parent of Chinese Simplified, which in turn is the neutral language parent of Chinese Simplified Legacy, which in turn is the neutral parent of Chinese Simplified Legacy (PRC). This function will find the closest neutral parent if it is not one already.
