CompositionSet

A CompositionSet is a unique set of Composition Hosts. It consists of 1 root composition container and N leaf CompositionHosts. A Composition set lifetime matches the lifetime of the leaf composition containers / hosts. A Composition set can be locked into memory forcing the lifetime of the set including the lifetime of the root composition container by calling CompositionSet.InitCompositionSet.

The CompositionHost represents and manages the object which makes up the entire next gen application stack.
