RelativeWidthsToVisibilityConverter

This converter is used to determine the visibility of a control based upon two or three values: a 'container' width, an 'ideal' width, and an optional ItemCollection. It was written to provide a way to only display an 'assistant' control (e.g. a dropdown item selector) when there are too many elements (e.g. document tabs) to be shown in the editor. If all document tabs can be shown, then the dropdown item selector will not be visible.

The optional parameter argument may be a Visibility value to specify how a control will be hidden (Visibility.Hidden or Visibility.Collapsed). By default, if it is determined that the return value is not Visibility.Visible, the Convert method will return Visibility.Hidden. This is done in order to preserve layout of companion visuals that may be arranged around the one using this converter. Note that one could effectively disable this converter by providing a value of Visibility.Visible to the parameter argument.

ValueConverterGroup

A value converter which contains a list of IValueConverters and invokes their System.Windows.Data.IValueConverter.Convert(System.Object, System.Type, System.Object, System.Globalization.CultureInfo) or System.Windows.Data.IValueConverter.ConvertBack(System.Object, System.Type, System.Object, System.Globalization.CultureInfo) methods in the order that they exist in the list. The output of one converter is piped into the next converter allowing for modular value converters to be chained together. If the System.Windows.Data.IValueConverter.ConvertBack(System.Object, System.Type, System.Object, System.Globalization.CultureInfo) method is invoked, the value converters are executed in reverse order (highest to lowest index). Do not leave an element in the Converters property collection null, every element must reference a valid IValueConverter instance. If a value converter's type is not decorated with the ValueConversionAttribute, an InvalidOperationException will be thrown when the converter is added to the Converters collection.
