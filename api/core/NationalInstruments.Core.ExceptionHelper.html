CatchGeneralExceptions

Do not use this property directly, use ShouldExceptionBeCaught(Exception).

Determines if general exception handlers should be in the code paths of execution. This is typically done for all products when not under a debugger. This lets us of inform users of internal errors in a more graceful manner while giving developers a good debug experience.
