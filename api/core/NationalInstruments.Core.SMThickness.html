Struct SMThickness

SMThickness type. SMThickness is similar to System.Windows.Thickness, except it is float-based instead of double-based. Also, only those methods on Thickness that we are using have been implemented in SMThickness, it is not a full-featured alternative; however, we can add methods as needed.
