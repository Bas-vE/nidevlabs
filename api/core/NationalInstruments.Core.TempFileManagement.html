CreateTempFile(String, Boolean, out String)

Creates a temporary file with the specified name in the NI temporary directory. Once the returned FileStream is closed, the OS will automatically delete the temporary file from disk (even if the process is terminated). The temporary file created is created with read/write access and no sharing permissions.

Declaration
public static FileStream CreateTempFile(string fileName, bool generateUniqueName, out string tempFile)
Parameters
Type Name Description
String fileName

Name of the temporary file to create. If null or empty, a GUID will be used to generate a unique name and the ".tmp" file extension will be used.
Boolean generateUniqueName

If true, will use a GUID to ensure a unique name in the temp dir. The generated name will have the form {fileBaseName}_{GUID}.{fileSuffix}
String tempFile

The path to the file that was created.

CreateTempFileFromExistingFile(String, out String)

Creates a temporary copy of a file with the specified path in the NI temporary directory. Once the returned FileStream is closed, the OS will automatically delete the temporary file from disk (even if the process is terminated).
