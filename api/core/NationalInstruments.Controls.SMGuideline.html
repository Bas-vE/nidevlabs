RelativeTo

Gets and sets the name of the guideline which this guideline is placed relative to. When using relative placement you can set the Percentage to be Nan in which case the desired size of the attached objects will be used as the distance this guideline is placed relative to the other guideline. If you use a non Nan margin then the RelativeTo position will essentially be added to the start margin of this guideline.
