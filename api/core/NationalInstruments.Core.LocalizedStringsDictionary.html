MakeDictionaryName(IEnumerable<String>, Int32, Int32)

Ensures the proposed name for a LocalizedStringsDictionary is suitable for use in XAML.

Declaration
public static string MakeDictionaryName(IEnumerable<string> elements, int maxKeyLength, int trailingElementsToRetain)
Parameters
Type Name Description
System.Collections.Generic.IEnumerable<String> elements

Elements used to compose the key. These will be separated by underscore characters.
Int32 maxKeyLength

The maximum allowed length for a dictionary key. Ignored if negative or zero. This value should be at least 32.
Int32 trailingElementsToRetain

The number of elements from the end of elements to retain at the end of the returned string.
Returns
Type Description
String

A form of the name that is suitable for use in XAML, e.g. as a value in LocalizerExtension.
Remarks

The strings in the elements enumerable are joined, using underscore as a separator to form string. If the string is too long based on the value passed in via a valid maxKeyLength, the string will be truncated to the given length, with a checksum of the unchanged result replacing the truncated portion of the string. If trailingElementsToRetain is > 0 and valid, the elements will be retained, if possible, if the key is too long. For example, if the unchanged key were: this_is_a_key_that_is_far_too_long_and_it_should_be_shorter_than_50_characters and the maximum length is 50, and no trailing elememnts were to be retained, the returned result would be something like: this_is_a_key_that6294bbe6c4c17001845399cb41dc2c5a If a maximum length of 50 is used, and in addition the trailing elements to retain is set to two, the result would be: this6294bbe6c4c17001845399cb41dc2c5a_50_characters If a maximum length and trailing element combination cannot retain the trailing elements, the request to retain trailing elements is ignored, and the normal truncation takes place. Finally, if the maximum length is exceeded, but too short to allow the hash to be used, InvalidOperationException will be thrown.

Exceptions
Type Condition
InvalidOperationException

Thrown if maxKeyLength is too short to allow a key to be created that contains a hash of the original key.
