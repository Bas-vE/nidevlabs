Class DispatcherPeriodicUpdateManager

This class manages a list of IPeriodicUpdateListener and makes sure these listeners are told to send change notifications. Two timers are used to perform the updates. The first fast timer runs at a high rate with a background priority. This timer will usually perform the notifications / update of the UI when it is not busy. When the UI is busy the fast timer will starve so we also use a slow timer which attempts to keep some updates going while leaving enough headroom in the UI thread to allow other operations.
