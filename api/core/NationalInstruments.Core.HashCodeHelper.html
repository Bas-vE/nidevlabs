CombineHashCodes(Int32, Int32)

Combines two hash codes into a new hash code. Uses a non-commutative and non-associative formula. Not suitable for hashing together elements of collection whose order doesn't remain fixed (e.g. set, dictionary).

ComputeStringHashCodeIgnoringCase(String)

Computes a case-insensitive hash code for the specified string using the invariant culture. This hash function is optimized for codes below 0xDF, but it will work for all strings. If you need a case-sensitive hash code, use string.GetHashCode() instead.
