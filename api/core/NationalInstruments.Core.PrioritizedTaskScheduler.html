PrioritizedTaskScheduler.BoostToken boostToken

The boost token that will be passed to ReturnToNormalPriority(PrioritizedTaskScheduler.BoostToken) at some time later. This boost token must be obtained through a call to CreateBoostToken().

CreateBoostToken()

Creates a boost token for use with the BoostPriority(AsyncTaskPriority, PrioritizedTaskScheduler.BoostToken) and ReturnToNormalPriority(PrioritizedTaskScheduler.BoostToken) APIs. This token can be passed to BoostPriority multiple times. Once it has been passed to ReturnToNormalPriority, it can no longer be used to boost a scheduler's priority.
