ContinueOn<T>(Task<Tuple<ActivityResourceAffinities, T>>, IScheduledActivityManager, AsyncTaskPriorityToken, String)

Causes the continuation following the usage of async/await to run on the activity manager. The continuation's affinity is determined late; i.e. by the task returning a desired affinity on completion. See remarks.

Declaration
public static ActivityAwaitable<T> ContinueOn<T>(this Task<Tuple<ActivityResourceAffinities, T>> task, IScheduledActivityManager activityManager, AsyncTaskPriorityToken priority, string description = "")
Parameters
Type Name Description
System.Threading.Tasks.Task<System.Tuple<ActivityResourceAffinities, T>> task

The task to be awaited.
IScheduledActivityManager activityManager

The activity manager used to marshal the continuation.
AsyncTaskPriorityToken priority

The AsyncTaskPriorityToken to be associated with the continuation.
String description

A debug description of what action the task is performing
Returns
Type Description
ActivityAwaitable<T>

An awaitable object used to marshal a continuation following a usage of the 'await' keyword to the activity manager.
Type Parameters
Name Description
T

type of the result of the continuation
Remarks

If the task is canceled or faulted, awaiter will throw an exception on the calling context. This means that the task never actually returns an affinity, so caller must be flexible about cancelable/unsafe tasks.
