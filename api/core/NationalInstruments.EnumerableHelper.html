MinAndMax(IEnumerable<Double>, out Double, out Double)

Return the min and max value found in an enumerable of doubles. If the sequence is empty, min will return PositiveInfinity and max will return NegativeInfinity. This is the only case in which min will be greater than max. Min and max may be equal if the enumerable contains all the same values.
