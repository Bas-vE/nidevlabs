Interface IScheduledActivityManager

The scheduled activity manager used to schedule and manage ScheduledActivitys.

Namespace: NationalInstruments.Core
Assembly: NationalInstruments.Core.dll
Syntax
public interface IScheduledActivityManager
Remarks

MEF is used to create an ScheduledActivityManager singleton. The singleton is accessed by way of Host.GetSharedExportedValueIScheduledActivityManager or Host.GetSharedExportedValueScheduledActivityManager

Remarks

There are several implementations of ScheduledActivity that might be useful:

  • BasicActivity -- Concrete, simple activity that will call a user supplied action
  • ScheduledActivity -- Abstract implementation of ScheduledActivity that most activities should have as an ancestor
  • EnumerableActivity -- Abstract implementation of ScheduledActivity that provides a nice syntax for yield return style work
