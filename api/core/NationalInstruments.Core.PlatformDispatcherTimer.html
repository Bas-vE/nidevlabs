Class PlatformDispatcherTimer

Platform-independent dispatcher timer. Wraps a timer that executes on a PlatformDispatcher.

Remarks

PlatformDispatcherTimer enforces thread affinity for operations that may modify the state of the object. Specifically, property setters, starting and stopping the timer must be done from within the thread upon which the provided PlatformDispatcher is running. Violations of this rule result in InvalidOperationException. Note that several properties, such as Interval, IsEmpty, et. al. are in the platform-specific implementations. This class also implements IDisposable, so that any abandoned timer will be stopped.
