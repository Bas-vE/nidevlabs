SetHitTestOverlay(PlatformVisual, Boolean)

Sets the HitTestOverlay property on a visual. Setting this property on a visual will cause the VisualTreeWalker hit testing methods to ignore the visual while hit testing, but the children of the visual will not be ignored. This does not affect the input event hit testing that WPF does when routing input events.
