CreateUniqueName(IEnumerable<String>, String, String)

Given a list of unique names, this function will check to see if the desired name conflicts with any existing names in the collection.

If it does conflict, it will product a new unique name that is derived from the desired name, appended with the specified uniquification, appended with the specified postfix.

If it doesn't conflict, the original desired name is returned with the postfix.

GetNextUniqueValue(IEnumerable<String>, String)

Gets the next integer which can be appended onto baseName and form a unique string within a sequence of strings. The integer returned will be the maximum for the set of matching base names such that N unique names can be constructed simply by incrementing the number returned.

GetUniqueNamesFromNumber(String, Nullable<Int32>, Int32)

Generates a number of string names, all beginning with a base name and with a underscore-plus-integer uniqification.

This function creates names without a string postfix.
