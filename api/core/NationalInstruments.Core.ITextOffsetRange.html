Contains(Int32)

Returns whether or not the offset is contained withing this ITextOffsetRange.

bool Contains(int offset)
Parameters
Type Name Description
Int32 offset

The offset to check.
Returns
Type Description
Boolean

true if the offset is contained in the range. An offset is considered to be contained by a range if it is greater than or equal to the StartOffset and it is less than the EndOffset.

Type Description
Boolean

true if the current ITextOffsetRange contains the other range. A range is considered to be contained if its start offset is greater than or equal to the start offset of this range, and its end offset is less than or equal to the end offset of this range.
