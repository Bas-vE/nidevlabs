WaveformDeltaTimeOverridePropertyName

This string name a particular ExtendedProperty of a Waveform type. This property stores a Double that overrides the usual computation of the delta time. Since infinity cannot be represented on the fundamental type of the delta time, all operations on a Waveform must check for this property before resolving the delta time value. The value may be any value, but at this time it is only used for positive or negative infinity.
