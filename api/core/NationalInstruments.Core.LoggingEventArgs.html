Remarks

Formats the logging event as, where [...] means it is optional: 

+{TimeOffsetFromStart.TotalSeconds}.{TimeOffsetFromStart.Milliseconds} {Source.FullName} >>[>>] {FullMessage}

The time is an offset from StartTime.
