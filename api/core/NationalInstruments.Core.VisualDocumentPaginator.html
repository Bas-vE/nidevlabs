Class VisualDocumentPaginator

Default paginator for a WPF Visual. Not sure why WPF doesn't provide one by default, but I suppose they want you to think about where your page breaks are. To use, you create with pagesize and margins (see comments in constructor on how to get from PrintDialog). Then set Content = Drawing of your visual. A drawing can easily be created from any visual with UIHelpers.CreateDrawingVisual
