Struct AsyncTaskPriorityToken

A token which represents the priority at which to run asynchronous operations. A Priority token can have a specific priority set on it and it can derive its priority from a AsyncTaskPriorityTokenSource. The priority will always be the Max of the specific priority and the priority of the AsyncTaskPriorityTokenSource if there is one.
