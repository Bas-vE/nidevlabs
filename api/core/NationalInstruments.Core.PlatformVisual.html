Struct PlatformVisual

This type provides a light-weight, cross-platform wrapper for platform-specific visual types. The properties and methods in this portion of the type shall provide reasonable implementations for all our supported platforms. Platform-specific extensions to this type are located in the respective platform-specific portions of the struct.
