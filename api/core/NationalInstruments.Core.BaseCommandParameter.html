UpdateFromCommandIfNeeded(ICommandParameter, ICommandEx)

Only update from the command if the command parameter does not have a value set. It is assumed that if the command parameter has a set value then it is taking precedence over the commands default values. This is especially important for popup menus where this code is reached after the command enablers are called, and the command enablers will only be called once.
