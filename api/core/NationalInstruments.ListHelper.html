RemoveFromEndWhere<T>(IList<T>, Func<T, Boolean>)

Removes all elements of a list that satisfy a given predicate, but removes from the end rather than from the beginning. This allows undo operations to simply add the items back to the end of the list in order to retain the original order of the items in the list. This version is also slightly more efficient than the RemoveWhere method since fewer remaining list items need to be renumbered.
