System.Func<CancellationToken, System.Threading.Tasks.Task<TResult>> func

An async delegate to be run by this activity. This action receives a CancellationToken that is signaled anytime the CancellationToken associated with this activity is cancelled. This can occur when the CancellationToken given to the activity at construction is cancelled, or when the ScheduledActivityManager is shut down. This CancellationToken is not the same as the given token, but it will be signaled when the given token is signaled.
