HandleOverflow(Number, NIType, NumericOverflow)

Coerce a number's left bits to within a given bit range by wrapping or saturation. This assumes rounding has already been applied, and only handles left bits. In wrap mode, for example, it is possible for this to return a positive value exceeding the specified range (due to extra right bits).
