Remarks

This method is trying to render FormattedText as close as TextBlock. However, there are two cases still different:

  1. Wrapping and Trimming text. e.g: FormattedText v.s. TextBlock | ASuperSu... | | ASuperSuper | | | | SuperLongWo | | | | rd |
  2. Multiple-line, no wrapping and trimming text. e.g: FormattedText v.s. TextBlock | FirstLin | | First... | | e | | Secon... | | Secon... | | Third... |
