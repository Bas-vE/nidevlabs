public static bool IsModifier(this NIType type)
Parameters
Type Name Description
NIType type

the type to check.
Returns
Type Description
Boolean

true if the type is a modifier, false otherwise.
Remarks

A modifier is any type that is a thin wrapper around the real type. It should (assuming no bugs) mean that all of the the "Is" test functions drill through modifiers to the underlying type. Typedefs and attributed types are modifiers. That means if you have, for example, an attributed type in your hand and you ask "IsDouble", it will look inside itself at its inner type and let the inner type answer the IsDouble question. It also SHOULD mean that all of the functions to access internals of the type work equally well on the modifier as on the real type (for example, if you have a modifier wrapping a class type, you can call GetFunctionNames without having to first get the underlying type).

The only "Is" functions that do not drill through, obviously, are those that check if a type is a modifier, such as IsTypedef(NIType), IsAttributed(NIType), and IsModifier itself.

Any new NIType functions should work the same when given a type or a modifier of that type unless explicitly noted otherwise.
