public class AttachedPropertyStorage
Remarks

This class is used by the extension methods in DynamicPropertyExtensions to implement external attachment of properties to objects. Currently, this implementation is hidden under those methods, and there is no way for a client to pick a different external attachment store. We might want to add this in the future if we want to support thread-safe vs. not or whatever. Right now, this is the most general implementation and does include thread safety.
