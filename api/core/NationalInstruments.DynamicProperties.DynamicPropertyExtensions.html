Remarks

First, we attempt to cast the object to IPropertyLocalStorage. If that works, we use that interface to attempt to get the property via TryGetValue(PropertySymbol, out Object). If that succeeds, we're done. If not, and the property symbol supports attachment (), we try to get the value of the property from the attached property store. If neither of those works, we return .

IsPropertySet(Object, PropertySymbol)

Returns true if a property is set on an object. If a property is not set, it will return a default value from GetPropertyValue(Object, PropertySymbol). You can un-set a property by calling ClearPropertyValue(Object, PropertySymbol).

Remarks

First, we attempt to cast the object to IPropertyLocalStorage. If that works, we use that interface to attempt to set the property via TrySetValue(PropertySymbol, Object). If that succeeds, we're done. If not, and the property symbol supports attachment (), we try to attach the property using an external store. If neither of those works, this method will throw.
