CalculateDamerauLevenshteinDistance(String, String)

Calculates a case-sensitive Damerau-Levenshtein distance between two strings. Used for non-exact string comparisons.

Declaration
public static int CalculateDamerauLevenshteinDistance(this string source, string target)
Parameters
Type Name Description
String source

The source string. Can be null or empty.
String target

The target string. Can be null or empty.
Returns
Type Description
Int32

A numerical distance measurement between the two strings.
Remarks

This is defined as the minimal number of operations needed to transform the source string into the target string, where an operation is defined as an insertion, deletion, or substitution of a single character, or a transposition of two adjacent characters.
