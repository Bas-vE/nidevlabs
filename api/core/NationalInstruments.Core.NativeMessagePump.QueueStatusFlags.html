Enum NativeMessagePump.QueueStatusFlags

These flags describe the status of a Win32 message queue.

Namespace: NationalInstruments.Core
Assembly: NationalInstruments.Core.dll
Syntax
[Flags]
public enum QueueStatusFlags

Fields

Name Description
QS_ALLEVENTS

A mouse, key, posted message, timer, paint, or hotkey message is in the queue.
QS_ALLINPUT

A message of some type -- any type -- is in the queue.
QS_ALLPOSTMESSAGE

A posted message is in the queue; this value is cleared if GetMessage or PeekMessage is called without filtering.
QS_HOTKEY

A hotkey message is in the queue.
QS_INPUT

A user input (mouse or key) message is in the queue.
QS_KEY

A key message is in the queue.
QS_MOUSE

A mouse move or button message is in the queue.
QS_MOUSEBUTTON

A mouse button message is in the queue.
QS_MOUSEMOVE

A mouse move message is in the queue.
QS_PAINT

A paint message is in the queue.
QS_POSTMESSAGE

A message has been posted to the queue.
QS_RAWINPUT

A raw input message is in the queue.
QS_REFRESH

A hotkey, key, mouse button, or paint message is in the queue.
QS_SENDMESSAGE

A message sent by another thread or application is in the queue.
QS_TIMER

A timer message is in the queue.
