IsSupportedUnderScope(IQualifiedScopeService, BindingKeyword)

Determine if a given envoy is supported under the target/scope. All targets are allowed to own and load any type of envoy. The idea of supported refers to other operations and how well they are handled. For example, it might be legal to display a document for a non-supported envoy. However, the document might behave differently when compared to a supported document. It might open in a read-only manor preventing any user modifications. It might lack functionality common with other document types such as palettes, colored wires, or an associated compiler.
