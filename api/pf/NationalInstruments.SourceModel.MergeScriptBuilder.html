Parameters
Type Name Description
XmlWriter writer

The XML writer to use for writing the merge script
Boolean writeMetadata

True if the source model feature set and merge data should be generated as part of the merge script and False if it should be omitted. You will generally only pass False if the merge script is going to be persisted as part of a larger file that already has a source model feature set and doesn't need any other meta data.
Remarks

Upon building the merge script, all Elements added and any of their descendants that implement the IMergeSource interface will be queried for additional elements to add to the merge script. This process is repeated until no new elements are added and is done such that the same element isn't added twice.
