If non-null, this filter is used to determine if a particular PaletteElement (and by implication, its children) should be localized.
System.Action<PaletteElement, LocalizedStringsDictionary> customAction

A custom action to execute when localizing a palette element. This is delegate is called after the standard parts of the palette element have been updated. See remarks.
Remarks

The standard behavior is to localize the default user-facing text and search terms (synonyms) for a PaletteElement. These standard properties are: Display, ShortName, InfoTip, HelpLink, and Synonyms. Any further customization that may be desired, such as the handling of the Target (e.g. a MergeScript) should be handled via the customAction delegate.

PopulateLocalizationDictionary(PaletteElement, LocalizedStringsDictionary, Predicate<PaletteElement>, Action<PaletteElement, LocalizedStringsDictionary>)

Populate entries in a string dictionary for palette elements. The default behavior handles standard display text. The palette element's children will also be added to the dictionary recursively.
