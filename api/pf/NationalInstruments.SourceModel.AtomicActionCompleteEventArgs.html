Class AtomicActionCompleteEventArgs

The EventArgs for the AtomicActionComplete event. The TransactionManager will continue to pump AtomicActionComplete events until no handler sets NeedsAnotherPass to true. At the end of a given atomic action, each pump of AtomicActionComplete will have Pass be one higher (starting with 0). NeedsAnotherPass can only be set to true.
