Type Description
ILockedSourceFile

The ILockedSourceFile for the newly created and added envoy. This object controls the lifetime of the in memory representation of the referenced file in memory. When the last outstanding instance of this object is disposed, the referenced file in memory will be released and eligible for garbage collection, and the envoy will need to be re-loaded in order to access the contents of the file.
Remarks

When this method is called, there is not necessarily a file on disk that is associated with the Envoy. Thus, implementers of this method must immediately call BeginLoadFromStream(Stream, EnvoyLoadOptions, PlatformDispatcher, Action<ILockedSourceFile, Exception>) to load the referenced file from the stream. The file may be given a storage path and saved at a later time.
