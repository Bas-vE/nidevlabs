Interface ISymbolFactory

A factory that can create PropertySymbols for dynamic properties while parsing XML.

Namespace: NationalInstruments.SourceModel.Persistence
Assembly: NationalInstruments.PlatformFramework.dll
Syntax
public interface ISymbolFactory
Remarks

When parsing XML, if we see an XML attribute with a fully-qualified XName (namespace + name), we assume that this is a dynamic property that the object itself doesn't know about. We therefore consult an external factory to create a symbol and parse that property. The PropertySymbol is what contains the serialization mechanism that we use.
