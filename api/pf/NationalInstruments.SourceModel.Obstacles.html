The weight of the obstacle being added. The weight value make sense only for Translucent obstacles.

If the obstacle to be added needs to be reversed. The reversed flag indicates if the left face must be interchanged with the right. This is usually set when we need to insert the whole diagram bounds or define a region within which the routing needs to take place.

The Id original owner in the model.
