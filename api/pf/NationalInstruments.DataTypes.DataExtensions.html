GetNIType(Object)

Get an NIType to represent the type of a piece of data. Since there isn't a one to one mapping between NITypes and value objects, you need to use this function with care. For instance, a UInt32 value object could correspond to either a UInt32 NIType or a SizedUInt NIType.
