Interface IDataSerializationFactory

Interface which represents a class which is capable of making other classes that know how to serialize or deserialize data types. This factory is datatype specific, so you should use DataEnumeratorFactory in order to get a reference to one of these datatype-specific factories.

Type Description
IDataDeserializer

A deserializer meant to read the given input, or null if this factory cannot handle the input.
Remarks

The validity check of the input may be a spot check, meaning the function may return a non-null value, yet the input is not correct. This was done in order to make import more efficient, doing as little reading in as possible. Check the output of the deserializer for null when using it, and if it returns null, you should continue to a new factory.
