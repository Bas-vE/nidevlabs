TargetsToCheck

Gets a collection of Envoys to check if there are Elements in the system that have been notified of export changes to the given Envoys, but the notifications have been deferred. If either DefinitionToCheck or ListenersToCheck is not null, this value must be null. If both DefinitionToCheck and ListenersToCheck are null, this data will instruct the linker to check all elements in the system that have pending notifications from any of the Envoys in this list.
