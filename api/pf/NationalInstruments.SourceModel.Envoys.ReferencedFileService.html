Class ReferencedFileService

A base implementation of IReferencedFileService for clients that save files to disk that do not match the source model persistence format. The Definition for these files, if any, will not be owned by a SourceFileBase, and will not be persisted.

System.Threading.Tasks.Task<ILockedSourceFile>

A System.Threading.Tasks.Task<TResult>representing the asynchronous load operation. The ILockedSourceFile associated with this Task is a hint to the lifetime of the loaded file. While this object is not disposed, the loaded file will remain loaded under most circumstances. In some situations, like refreshing a file from disk, the file maybe be unloaded even if this object is not disposed. To prevent the file from being unloaded, you must hold a read lock on the Envoy owning the file, which can be acquired through AcquireModelReadLock().
