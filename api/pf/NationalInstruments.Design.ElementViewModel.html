Class ElementViewModel

Base class for Model Element View Models. The base implementation watches for property changed notifications and integrates with the InUserModification system to manager when property changed notifications are sent to the visuals themselves. This makes sure that only properties that actually changed during a Revert / Commit user change are seen by the visual objects.

public virtual IEnumerable<Adorner> GetHardSelectAdorners()
Returns
Type Description
System.Collections.Generic.IEnumerable<Adorner>

The additional adorners to be displayed when the item is hard selected.
Remarks

This method should only return the additional adorners that are not already included in the collection returned by GetSoftSelectAdorners(). When an item is hard selected all of the soft select adorners will transition to their hard selected appearance.

Resize adorners are added via IResizableItem before GetHardSelectAdorners is called. Users must add any other default adorners that they want displayed to this list before returning. Returning null or an empty collection means that no adorners will be displayed (excluding resize).
