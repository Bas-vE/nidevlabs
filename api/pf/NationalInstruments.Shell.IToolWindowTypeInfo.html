InitialFloatingLocationXOffset

Defines the X value of the initial floating location of the window in terms of the application window. If the value would put the window too far to the left or right, it is clipped so as to ensure that the initial window location is bound to the application window space. This is an offset, and works from the selected value of InitialFloatingLocation.
