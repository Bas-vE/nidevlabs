Class FileImporter

Class to handle importing files from disk to a Definition. Files are specified by FileStreamInfo and are then copied to the appropriate folder on disk corresponding to the imported file's given parent envoy.

Remarks

This class does not utilize FileStream property in FileStreamInfo class. If the caller chooses to open FileStream, the caller is responsible for closing it.
