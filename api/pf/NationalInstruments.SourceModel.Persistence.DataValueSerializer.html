GenerateDataValueCookie(IElementPropertyGenerator, XmlWriter, Object)

This function allows you to serialize a cookie that represents an object in memory instead of fully serializing out the entire contents of the object. This cookie can then be used to lookup the object when parsed instead of parsing the entire object again. This is an optimization that can only be used for copy/paste within the same process.
