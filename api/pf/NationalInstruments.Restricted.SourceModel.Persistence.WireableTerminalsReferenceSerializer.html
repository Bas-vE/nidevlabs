Class WireableTerminalsReferenceSerializer

This Serializer is used to handle Terminals properties on Element XmlElements, i.e. Terminals="1179, 1181=1207, 1182=1190". The behaviour of Generate is dictated by the desired NationalInstruments.Restricted.SourceModel.Persistence.WireableTerminalsReferenceSerializer.WireableAttributes as well as the ModelGeneratorContext, which we get from the generator. The existing code in XmlModelGenerator is not structured well for reuse, so I provided a similar implementation here.
