Class ToolWindowEditSite

Provides callback methods for tool windows in the editor. Each tool window is provided with an implementation of this interface to use as needed. It should not be shared by other tool windows. The interface's lifetime matches the toolwindow. Get a EditSite for your tool window by calling GetEditSite().
