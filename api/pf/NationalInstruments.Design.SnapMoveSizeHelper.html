Class SnapMoveSizeHelper

Helper class which helps with moving and sizing elements with the following features:

  • Snapping to Grid
  • Snapping to the edges of sibling elements (while showing snap lines)
  • Resizing while staying on the grid
  • Resizing while maintaining aspect ratios
