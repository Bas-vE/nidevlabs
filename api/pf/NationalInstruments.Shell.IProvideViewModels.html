Interface IProvideViewModels

Interface used to define a factory to create ViewModels for a given Element derived class.

MEF Metadata requirements: [Export(typeof(IProvideViewModels))]

Set: [ExportMetadata("PresentationTypes", "Namespace Name of Model Types Presentation Supports / Facade Root Name")] This should be the sub-string of the presentation types that these resources are for. If this attribute is not present or is empty, these resources will be added before any diagram elements are bound
