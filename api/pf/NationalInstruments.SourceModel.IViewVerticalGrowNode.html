TopMargin

The area at the top of the grow node that is used as a space for the terminals that don't belong to a chunk- only fixed terminals. Note that if TopMargin is 0, we can still have fixed terminals because the minimum chunk count might be >= 1. The offsets of the fixed terminals never change.
