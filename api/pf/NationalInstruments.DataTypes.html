FixedPointHelperFunctions

Helper class to define limits of fixed-point type parameters and provide various validation utilities. The parameter limits are designed such that all values of any fixed-point type with word length of 53 bits or less should be exactly representable by a double. Values with word lengths greater than 53 bits will lose precision, but will be correct to 53-bit accuracy without overflowing or underflowing.
