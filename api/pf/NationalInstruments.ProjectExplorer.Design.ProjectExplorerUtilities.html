GetProjectItemInfo(Envoy)

Returns the IProjectItemInfo service on a give envoy. This interface is implemented by envoys that want to show up in the Project Explorer. Call this extension method instead of GetProjectItemViewModel() if you don't need the view model, rather just need to query information.

LoadProjectInformationInBackground(ICompositionHost, String, out ProjectInformation)

Loads the project information for the project at the given path. Uses the host to create a UniversalParser and check the project type. The caller must ensure that the projectPath is valid. If projectPath is located on a network drive, this function may take several minutes to evaluate, and should be called in the background.
