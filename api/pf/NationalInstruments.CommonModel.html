DataRangeComparer

IComparer interface than can be used to sort DataRange objects. This differs from DataRange.CompareTo in that you can specify the NIType you expect the DataRange to represent. Any DataRanges of a different NIType will be considered greater than DataRanges of the expected NIType. DataRanges with NITypes different than the expected NIType will then be sorted based on their ToString value.
