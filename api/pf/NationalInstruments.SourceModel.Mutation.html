NamespaceRenameMutation

General helper class for changing the name of a namespace. It's assumed the old namespace name is no longer used. It's also assumed that no property values contain the namespace name being removed. If that's not true, you'll need to write a custom mutation to handle this. The one exception to this is for the SourceModelFeatureSet which is updated to reflect the new namespace name.

UpdateNamespacesMutation

Mutation which updates namespaces. This can take be adding a namespace where previously one did not exist or changing/updating an existing one. For the later case, this can take two forms: 1) Matching the entire previous qualified name and replacing it. 2) Matching the prefix (namespace) portion of the qualified name, and replacing only that portion.

VirtualInstrumentTerminalIdMutation

General helper class for any VI mutation where a terminal changed locations or was renamed on the connector pane.
