ExposeVerticalChunkCountWithoutCreatingTerminalsProperty<TOwner>()

Special purpose property symbol for linked context propagation. This is needed because when an existing vertical grow node is grown, we'll see that the vertical chunk count was changed as well as new terminals were inserted. We want to propagate the terminal additions via either the vertical chunk count property or the inserts of the the terminals themselves, but not both. In this case, setting of the vertical chunk count won't trigger the insertion/deletion of terminals since we know they'll eventually get added/deleted through the normal propagation mechanism.
