Interface IDocumentTypeInfo

Interface which represents the metadata for a particular document type. MEF generates implementations of this interface for each IDocumentType registered with the system. The values of the parameters are set from the export meta data of the file type.

AutoCreatesProject

Specifies whether or not the document type can auto create a project and/or name scope. If the current project is hidden or there is no project open or the project does not contain a name scope, one will be automatically created when the document gets created. If it cannot auto create a project, then the document cannot be created unless there is an existing non-hidden project containing at least one name scoping envoy. This is a UI-only level feature. This will dictate whether or not the item is creatable if there is no project loaded and it will be suffixed with "New DocType and Project"
