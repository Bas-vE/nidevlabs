MakeAbsoluteQualifiedName(Definition)

Makes an absolute QualifiedName to the definition. The absolute QualifiedName can be used to find the object from any location within a hierarchy of IQualifiedScopes.

Declaration
public static QualifiedName MakeAbsoluteQualifiedName(this Definition definition)
Parameters
Type Name Description
Definition definition

The IQualifiedTarget for which the absolute name is requested.
Returns
Type Description
QualifiedName

The absolute QualifiedName.
Remarks
  • In order to make an absolute QualifiedName every ancestor IQualifiedScope of the object must also be a IQualifiedTarget with a non null name.
  • This is safe to call from outside the Host thread as long as getting an incorrect name is okay. i.e.: This method will not crash if changes are synchronously made in the UI thread.
