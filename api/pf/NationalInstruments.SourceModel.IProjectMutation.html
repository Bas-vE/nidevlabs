Interface IProjectMutation

Interface that is used to define a project mutation. These mutations are executed once the project is loaded. The DateAdded property will be used to define the order of execution for these types of mutations.

Namespace: NationalInstruments.SourceModel

Properties

DateAdded

Returns the date/time the mutation was introduced. This is meant to be a coarse mechanism for determining the order in which IProjectMutations should execute.
The mutations with the oldest date will be run first.

NumberOfMutationSteps

The total number of steps the mutation will perform. This property will be ready once the method has completed. Each step of the mutation will be reported through the IShowProgress interface that is passed to the MutateAsync(Project, IProgress<IShowProgress>). For instance if the value of this property is X, then it should be expected that CurrentProgressAmount will be set to X once the mutation is completed.
