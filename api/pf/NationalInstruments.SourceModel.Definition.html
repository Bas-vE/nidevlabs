LockForScripting()

Locks this definition for scripting. Locking for scripting prevents any other context from transacting on the model and allows the locking context to have read access to the model without explicitly acquiring the read lock. Note: This can currently only be performed for the UI thread. Eventually we will allow background activities to lock for scripting. When we do this the definition will be locked for the current activity.
