ComputeEuclideanDistance(INode, INode)

Computes the euclidean distance between the given source node and destination node. The distance between the center of the source node and center of the destination node is calculated and the distance from center of nodes to their boudaries is subtracted. This provides the exact distance between the rectangles and thus the nodes are considered as rectangular objects and not just pointed particles. Note - ( To-do ) - Have to check what happens when rectangles overlap.

ComputeRectangleOverlap(SMRect, SMRect, out Vector)

Computes the overlap for rectangles. If there is a complete overlap of one over another, a (0,0) vector is returned. If there is no complete overlap and if there is a partial overlap, the overlap distance is returned If there is no overlap, None is returned with a (0,0) vector.
