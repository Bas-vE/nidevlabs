Class XmlModelParserHelpers

This class is home for static methods that might be useful in more than one place, that relate to using the modeling side of the SourceModel, and that don't have a natural home in another class. The purpose of this class is to encourage factoring out of duplicated code. There is also a similar UIHelpers class.
