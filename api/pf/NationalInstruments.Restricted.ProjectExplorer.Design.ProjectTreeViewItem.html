Class ProjectTreeViewItem

This is our custom TreeViewItem control that is used by PREPMETRO to workaround its inability to bind properties in the xaml style/setter sections. This custom TreeViewItem control programmatically binds properties we are interested in. NOTE: This function could fail if 2 or more items share a name in the visual hierarchy.
