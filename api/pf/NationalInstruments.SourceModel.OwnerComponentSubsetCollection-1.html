Class OwnerComponentSubsetCollection<T>

This class provides a transparent, low-space-overhead view of a subset of another IList<T>.

Usage is subject to the following caveats:

  1. If multiple sub-views of the same list are created, then the inclusion predicates of each should not result in any overlap. That is, if included(x) is true for some element x in sub-view V of list L, then included(y) should not be true for any y in a different sub-view of L.
  2. The implementation of the sub-view uses very few assumptions about the underlying list, so some list operations are O(n) where they might otherwise be O(1).
  3. The sub-view is only intended to reflect the present state of its underlying list at any given time. Any post-hoc notifications of changes to the underlying list, such as insertions at or removals from a particular index, cannot in general be translated to information about a sub-view of the list.
