Interface IFileTypeManager

Describes an object responsible for managing all exported file types in the system. Is able to find IFileType and IFileTypeInfo implementations for all exported file types in the system.

Namespace: NationalInstruments.SourceModel.Envoys
Assembly: NationalInstruments.PlatformFramework.dll
