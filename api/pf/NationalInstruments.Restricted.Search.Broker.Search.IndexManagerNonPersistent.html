Class IndexManagerNonPersistent

NonPersistent IndexManager manages the NonPersistent IndexGenerator instances to build index and keep index up-to-date. It is a singleton class so there is only one copy of index data for each category throughout the application. It creates IndexGenerator instance only when searching that category for the first time.
