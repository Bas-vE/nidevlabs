PaletteItemViewModelBase.Target
Remarks

This target property contains the merge script of palette item, used for drag and drop. This overriding Target property allows us to modify the merge script from Target="%1" to Target="Function.gvi" in view model layer, without changing the "%1" in palette element merge script.

Because if the merge script Target="%1" in PaletteElement is changed to Target="Function.gvi" after the first new document method call is dropped, the subsequent new document method call cannot find and replace "%1" in PaletteElement merge script, the subsequently dropped method calls will always be the first method call merge script.

In short, for rightclick and docked palette under split view, when creating its palette item view model, we expect the merge script in Target should always contains Target="%1", not changed by the previous new document method call drag and drop.
