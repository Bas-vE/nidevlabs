DisposeForCloseDocument(ILockedSourceFile, Boolean)

When closing a document, dirty and non-dirty documents must be handled differently.

  • When a dirty document is closed, the user explicitly said to not save the document. Because of this, we do not want the RecentlyAccessedSourceFilesManager to keep the referenced file in memory.
  • When a non-dirty document is closed, it is okay for the RecentlyAccessedSourceFilesManager to keep the referenced file in memory.
