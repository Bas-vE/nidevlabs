Enum DiagramZIndex

Enum that specifies the z-order rendering behavior of an item in a diagram.
This enum should be used when needing to specify the z-order of an item. Literal values should not be used. If the choices are not sufficient, new ones can be added, but make sure they are approved by the Platform Framework architect.
