AdjustPosition(GuidelineModel, Double, Double, Double)

A helper function to adjust the position of a Guideline. Depending on this guidelines IsFixed and RelativeStart values it may set StartMargin, EndMargin or Percentage. This code was origionally located in GuidelineToolViewModel but was factored here so that it could be used by both the tool view model, and the GuidelinePanelBatchRule.
