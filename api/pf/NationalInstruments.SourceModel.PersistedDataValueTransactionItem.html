Class PersistedDataValueTransactionItem

A property transaction item for data values representing NIType values where the value for the property is persisted in some other storage (typically to a file) rather than being retained in memory. This is typically only used for very large objects where we don't want to have multiple instances of the objects that aren't currently being used eating up lots of memory. Instead we're willing to make the performance trade off of serializing them to some temporary storage and then inflating them again if undo/redo requires it.
