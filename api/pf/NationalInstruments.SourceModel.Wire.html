public abstract class Wire : Connectable, IPropertyLocalStorage, IParsable, IQueryService, IViewElement, IMakeSpace, IMergeSource, IWireAlgorithmInfo
Remarks

A wire is at heart a list of WireJoints, stored in _joints. This is the core data structure of the Wire. Each WireJoint in turn has its own joint list, which are the joints to which this particular joint is connected. None of these lists are ordered. You walk a wire by starting at some joint and recursively enumerating each joint's joint list.
