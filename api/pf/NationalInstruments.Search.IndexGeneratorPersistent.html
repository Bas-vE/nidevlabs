Class IndexGeneratorPersistent

Persistent IndexGenerators mainly deals with resource on disk that has a huge index size or takes a long time to check and update the index. The index will NOT check if it is up-to-date before search. The index is updating by a background thread or a separate process. Each implementer should provide MEF meta before the definition.
