Interface ICustomizeTerminalVisuals

Interface which provides customization for the appearance of terminals on a node. The Visual Element/ViewModel for a node can implement this interface to provide custom terminal visuals for it's nodes, or any parent view model can implement and support for children (who don't directly implement themselves)

If there are no terminal providers for a node then a DataTemplate resource named TerminalVisual will looked for and used for the visual of the terminal.

If a terminal visual cannot be found then a ellipse will be used.
