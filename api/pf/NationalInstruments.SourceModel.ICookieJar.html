Interface ICookieJar

Interface to the current cookie management entity. Cookies are reference counted IDisposables. The CookieJar reference counts the Cookies. The CookieJar is a singleton in the MEF export system. Use GetSharedExportedValue or [Import] to get the document manager from MEF.
