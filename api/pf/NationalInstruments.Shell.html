SnapEase

This is a useful easing function for making things snap in place. It acts dramatically different based upon this objects value of EasingMode. If EasingMode == EaseIn, then the animation will stay still and then move into the final destination on the last frame. If EasingMode == EaseOut, then the animation will move into its final destination on the first frame and then stay still for the remainder of the animation. If EasingMode == EaseInOut, then the animation will snap to a position half way when the animation time is half over and then snap to the final destination on the last frame.

StartupHelper

Provides a performance boost to cold application launch by pre-loading binaries in a background thread. The procedure to use is as follows:

  1. On an installed image run the application with the -train command line argument. Then go through the most common startup scenario and exit the application. This will generate the assemblyPreLoad.txt file in local app data directory of the application. Then install this file with the .exe.
  2. The installer needs to run the PostInstallPrep.exe application as part of post install. This will ngen the application and process the assemblyPreLoad file and generate the preload.txt file again in the local app data directory.
  3. At normal application launch the preload.text file will be processed in a background thread and load all of the binaries which will help launch performance on computers with traditional, non SSD, disks.

MEF Metadata requirements: [Export(typeof(IProvideDiagramPresentation))]

Set: [ExportMetadata("DictionaryType", "Qualified Class Name of Document Model Definition Type")] When defining resources for a DocumentEditor or Global Resources These resources are added at the Root Document Editor and are available to all editors of the document type Applications resources are added to the application during document binding

[ExportMetadata("DictionaryType", "Qualified Class Name of DocumentEditControl")] When defining resources for a specific DocumentEditControl, specify the class name of the document edit control.

[ExportMetadata("PresentationTypes", "Namespace Name of Model Types Presentation Supports / Facade Root Name")] This should be the sub-string of the presentation types that these resources are for. If this attribute is not present or is empty, these resources will be added before any diagram elements are bound. This can be used to delay loading of a resource provider until one of its resources is added.
