Class EnvoyManagerFileTypeDeprecated

This class is deprecated in favor of EnvoyManagerFileType. This class creates envoy managers with the old model format as described below.

Used to properly implement AddNewEnvoy(Element, EnvoyCreateInfo) for an EnvoyManagerFile source file. When creating an envoy manager, source file, the created model should look something like: NameScopingEnvoy - The parent element passed. NOTE: This should be the NullTarget. SourceFileReference - This is the envoy that is created SourceFile EnvoyManagerFile (Definition) EmbeddedDefinitionReference Definition that is bound by the exported ModelDefinitionType
