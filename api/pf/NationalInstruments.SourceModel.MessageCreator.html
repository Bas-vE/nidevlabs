Class MessageCreator

Class that can create a message once the Element is resolve on which it should be set. Messages are often intended for the terminal of a node, and that resolving is done by the node. NEVER EVER STORE THIS CLASS, YOU MUST ASK FOR A MESSAGE AND STORE THAT!
