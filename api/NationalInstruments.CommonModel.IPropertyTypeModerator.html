Interface IPropertyTypeModerator

Controls wishing to adapt at all during type prop must present an object implementing this interface through the interface IProvidePropertyBindingBehavior below. The set of queries in this interface is all the information type prop needs to know from controls that have adapting-type properties.

Given a property, returns a list of other properties on the same control whose types "depend" on the first's, i.e. if the first property changes type and the control adapts to it, what other properties would have their types changed.
