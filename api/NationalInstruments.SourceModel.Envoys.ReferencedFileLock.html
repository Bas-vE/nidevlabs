Class ReferencedFileLock

A class for managing the lifetime of a referenced file. Call Acquire(IReferencedFileOwner) to obtain an instance of ILockedSourceFile for the given file owner. Once all ILockedSourceFiles for the given file owner have been disposed, the referenced file can be unloaded.
