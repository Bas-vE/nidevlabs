AttributeName(UIModel, PropertySymbol)

The default implementation of VisitProperty uses this method to determine the XAML attribute for a particular PropertySymbol. By default, only Left, Top, Width, and Height are supported; derived classes can override this to map attributes for models that they care about. Returning null from this method means writing a particular PropertySymbol as a XAML attribute is not supported.
