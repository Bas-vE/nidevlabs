Class SnapEase

This is a useful easing function for making things snap in place. It acts dramatically different based upon this objects value of System.Windows.Media.Animation.EasingFunctionBase.EasingMode. If System.Windows.Media.Animation.EasingFunctionBase.EasingMode == System.Windows.Media.Animation.EasingMode.EaseIn, then the animation will stay still and then move into the final destination on the last frame. If System.Windows.Media.Animation.EasingFunctionBase.EasingMode == System.Windows.Media.Animation.EasingMode.EaseOut, then the animation will move into its final destination on the first frame and then stay still for the remainder of the animation. If System.Windows.Media.Animation.EasingFunctionBase.EasingMode == System.Windows.Media.Animation.EasingMode.EaseInOut, then the animation will snap to a position half way when the animation time is half over and then snap to the final destination on the last frame.
