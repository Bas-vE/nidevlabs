Class WeakActionCollection<TParameter>

A repository for commands associated with a command owner where the lifetime and validity of the command is tied to the lifetime of the owner. This repository is not thread safe. If thread safety is required, users of the repository need to provide their own.

The owner of the command being registered. The command will be considered valid and executable for the duration of the owner's lifetime.
System.Predicate<TParameter> canExecute

A delegate that will be called to determine whether the execute delegate can be called.
System.Action<TParameter> execute

A delegate that executes the logic of the command.
Remarks

Currently, only a single command per command owner is supported. If more than one command is registered, only the last command will be available for execution.
