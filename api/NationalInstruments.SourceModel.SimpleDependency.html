Class SimpleDependency

Basic Dependency class. It provides three basic capabilities: 1.) You can specify single Action to call that will adapt caller to current state of dependency. This action is invoked for resolve, rename, and property changes of dependency.

2.) You can specify Function for returning the specific IDependencyTargetExport that should be used for this dependency. Only envoys that return a non Null value from this function will be allowed to resolve to this dependency.

3.) You can override capability two and specify that any Envoy that matches by name alone is allowed to resolve the dependency.
