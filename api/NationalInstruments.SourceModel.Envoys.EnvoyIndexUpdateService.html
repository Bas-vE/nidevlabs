Class EnvoyIndexUpdateService

Envoy service for registering, unregistering, and detecting Envoy Index information. Here are some useful definitions when thinking about the Envoy Indexes:

  • KindAndKey - A pair of strings used to map to one or more envoys.
  • Envoy Index - A KindAndKey that is mapped to an envoy.
  • Envoy Indexes - A collection of EnvoyIndex items. The collection might be project wide, envoy manager wide, or source file wide.
  • Envoy Index Hints - A collection of KindAndKeys specifying what Envoy Indexes might be found in an EnvoyManager. This does not return the actual envoys but is instead just what KindAndKeys can be found. For more information on these hints, NationalInstruments.Restricted.SourceModel.Envoys.IEnvoyManagerEnvoyIndexHints.
