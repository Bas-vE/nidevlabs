Class PseudoPathStorageManager

A base class for a StorageManager that implements a pseudopath. Note that pseudopaths define their own internal syntax, including but not limited to their scheme:// identifier.

LastWriteTime(String)

Returns a DateTimeSignature object representing the date and time the specified file or directory was last written to.

Declaration
public override DateTimeSignature LastWriteTime(string path)
Parameters
Type Name Description
System.String path

The full path name including directory and file name
Returns
Type Description
DateTimeSignature

The DateTimeSignature representing the date and time the specified file or directory was last written to. If file does not exist, returns Empty.
Overrides
StorageManager.LastWriteTime(String)
Remarks

The precision of timestamps change in multiple different ways, e.g. the timestamp is rounded up to the next even second when files are copied from NTFS to FAT as well as zipping and unzipping files causes the timestamps to be rounded up like this. See https://support.microsoft.com/en-us/kb/127830 for more information. Files installed from the NI Package Manager may chop off milliseconds from the timestamp when installing.

Because of this, the DateTimeSignature class overrides equality to compare two timestamps within 2 seconds and disallows users from comparing them exactly to consider them equivalent. This avoids having to recompute caches, e.g., when the file contents were not truly changed.
