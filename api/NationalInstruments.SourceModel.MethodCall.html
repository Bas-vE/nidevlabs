This function is typically only called during semantic analysis reflection for a generic method call when an additional output terminal needs to be added. This happens when the reference type of the method call goes from being a closed generic type to an open generic type. In rare situations it may also be called when we have resolved but the layout triggered by that resolve has not completed by the time we reflect types. (CAR 562520)
