DeleteAsync(IEnumerable<Envoy>, IEnumerable<String>, ProjectExplorerViewModel)

NOTE: Pass in the edit site instead of the project explorer whenever possible so the UI will be blocked during the async work. Deletes envoys and their files on disk, and deletes the files on disk at paths. Creates a transaction and adds undo/redo support. Also makes sure that the project items for the files/folders being deleted are removed. This function does no prompting.
